ВИДЕО 2.1 — СМЕШИВАНИЕ ЦВЕТОВ
Duration: 25 min | Loop: 3 min × 8 | BPM: 68
Концепция
Два цветных круга медленно плывут друг к другу. Сталкиваются → сливаются → образуется новый цвет. Результирующий круг плывёт обратно и снова делится на исходные.
Все комбинации (каждый блок — другая пара):

	Красный + Жёлтый
	Оранжевый
	2
	Синий + Жёлтый
	Зелёный
	3
	Красный + Синий
	Фиолетовый
	4
	Красный + Белый
	Розовый
	5

	Голубой
	6
	Жёлтый + Белый
	Кремовый
	7
	Три цвета — Красный+Синий+Жёлтый
	Коричневый → взрыв радуги
	8

	Финальный микс
	Детальный цикл (90 сек):
Фаза 1 — ПОЯВЛЕНИЕ (0–15 сек)
* Левый круг появляется из левого края (150px)
* Правый круг появляется из правого края (150px)
* Оба медленно плывут к центру
Фаза 2 — СБЛИЖЕНИЕ (15–40 сек)
* Круги приближаются — скорость постоянная: 3px/сек
* На расстоянии 50px: начинают слегка деформироваться (притяжение)
* На расстоянии 20px: пауза 2 сек — напряжение!
Фаза 3 — СЛИЯНИЕ (40–55 сек)
* Круги касаются → начинают перекрываться
* Зона перекрытия: смешанный цвет
* Полное слияние: один большой круг (180px) нового цвета
* Вспышка яркости на 0.3 сек
* Новый цвет "расцветает" от центра наружу
Фаза 4 — РАСЦВЕТ (55–75 сек)
* Результирующий круг медленно пульсирует (показывает себя)
* Цвет насыщается до максимума
* Держится 20 сек
Фаза 5 — ДЕЛЕНИЕ (75–90 сек)
* Большой круг начинает делиться (из видео 1.4)
* Разделяется на два исходных цвета
* Они плывут обратно к своим краям
* Цикл начинается снова
Визуальные детали:

* При слиянии — два звука становятся аккордом
* Результирующий цвет = новая нота (интервал между исходными)

ВИДЕО 2.2 — ПРИЛИВ ЦВЕТА
Duration: 25 min | Loop: 4 min × 6 | BPM: 55
Концепция
Экран медленно заполняется одним цветом — как морской прилив снизу вверх. Потом цвет уходит. Приходит другой цвет — с другой стороны. Полностью без фигур — только цвет и движение цветовой границы.
Цикл цветов:
Красный → Оранжевый → Жёлтый → Зелёный → Синий → Фиолетовый → снова Красный
Детальный цикл (4 мин):
Фаза 1 — ПРИЛИВ (0–60 сек)
* Красная граница появляется снизу экрана (y=1080)
* Медленно поднимается: 1080px → 0px за 60 сек
* Скорость постоянная: 18px/сек
* Верхняя часть экрана: тёмно-синий фон
* Нижняя: красный цвет
* Граница: мягкая волна (не резкая прямая)
Фаза 2 — ПОЛНЫЙ ЭКРАН (60–90 сек)
* Весь экран красный
* Небольшая пульсация (±3% яркости)
Фаза 3 — ОТЛИВ (90–150 сек)
* Красный уходит вверх: 0px → 1080px за 60 сек
* Снизу открывается тёмный фон
Переход к следующему (150–180 сек):
* Тёмный экран 15 сек
* Оранжевый начинает подниматься — но с ЛЕВОЙ стороны
* Каждый цвет приходит с новой стороны: низ → лево → верх → право → низ...
Вариации направления по блокам:
* Блок 1: снизу вверх (прилив)
* Блок 2: слева направо (ветер)
* Блок 3: сверху вниз (дождь)
* Блок 4: справа налево
* Блок 5: диагонально (45°)
* Блок 6: концентрически от центра
Граница цвета — детали:

  const wave = Math.sin(x * 0.02 + frame * 0.05) * 15; // амплитуда 15px
  return baseY + wave;
};

// Переходная зона: 30px градиент (тёмный → цвет)


Музыка:
* Каждый цвет — своя нота. Красный=С3, оранжевый=D3, жёлтый=E3...
* Нота держится пока цвет на экране
* При переходе: плавное glissando вверх-вниз
* Почти тишина при пустом экране

ВИДЕО 2.3 — РАДУГА СОБИРАЕТСЯ
Duration: 25 min | Loop: 3 min × 8 | BPM: 65
Концепция
Семь цветных дуг появляются по одной и складываются в радугу. Потом разлетаются обратно. Снова собираются. Каждый раз — новый порядок появления.
Детальный цикл (90 сек):
Фаза 1 — ПУСТОЕ НЕБО (0–5 сек)
* Голубой-серый фон
* Тишина
Фаза 2 — ПОЯВЛЕНИЕ ДУГ (5–60 сек)
* Каждые 8 сек — появляется одна дуга:
   * Красная (внешняя, r=340px) — слева вырастает к центру
   * Оранжевая (r=310px)
   * Жёлтая (r=280px)
   * Зелёная (r=250px)
   * Голубая (r=220px)
   * Синяя (r=190px)
   * Фиолетовая (r=160px, внутренняя)
Механика появления одной дуги:
Дуга рисуется как будто художник ведёт кистью от левого конца к правому.
Длина дуги: 0% → 100% за 6 сек (easing out)


Фаза 3 — ПОЛНАЯ РАДУГА (60–75 сек)
* Все 7 дуг на экране
* Радуга слегка пульсирует — яркость ±8%
* Снизу радуги: зелёная трава (несколько пикселей)
Фаза 4 — РАЗЛЁТ (75–90 сек)
* Все дуги одновременно "улетают" вверх и за горизонт
* Исчезают в разные стороны (красная — вправо, фиолетовая — влево)
* Или: тают (opacity 1→0 за 10 сек)
Вариации порядка появления (по блокам):
* Блок 1: снаружи внутрь (красная первая)
* Блок 2: изнутри наружу (фиолетовая первая)
* Блок 3: случайный порядок
* Блок 4: парами (красная+фиолетовая одновременно, потом оранжевая+синяя...)
* Блок 5: быстрее в 2 раза
* Блок 6–8: комбинации
Специальный момент (блок 7):
Радуга собирается → все дуги начинают двигаться → закручиваются в спираль → спираль разворачивается снова в радугу.
// Рисование дуги (arc)
const drawArc = (
  cx: number, cy: number, radius: number,
  startAngle: number, endAngle: number,
  progress: number, // 0-1 прогресс рисования
  color: string, strokeWidth: number
) => {
  const currentEndAngle = startAngle + (endAngle - startAngle) * progress;
  // SVG arc path
  return `M ${cx + radius * Math.cos(startAngle)} ${cy + radius * Math.sin(startAngle)}
          A ${radius} ${radius} 0 ${currentEndAngle - startAngle > Math.PI ? 1 : 0} 1
          ${cx + radius * Math.cos(currentEndAngle)} ${cy + radius * Math.sin(currentEndAngle)}`;
};


Музыка:
* Каждая дуга появляется с нотой своего цвета
* 7 нот = 7 дуг = 7 ступеней гаммы (до ре ми фа соль ля си)
* Полная радуга = полная гамма = красиво!

ВИДЕО 2.4 — ФРУКТЫ ПЕРЕКРАШИВАЮТСЯ
Duration: 25 min | Loop: 4 min × 6 | BPM: 72
Концепция
Группа фруктов стоит в ряду. Волна цвета проходит слева направо. Каждый фрукт при прохождении волны меняет цвет на следующий в радуге. Как домино — но цветное.
Структура блоков:
Блок 1 (0–4 мин): ПРОСТАЯ ВОЛНА — 3 ФРУКТА
Стартовая позиция:
* Яблоко (красное) | Апельсин (оранжевый) | Банан (жёлтый)
Волна цвета идёт слева направо:
* Яблоко становится ОРАНЖЕВЫМ → пауза → ЖЁЛТЫМ → ЗЕЛЁНЫМ → СИНИМ...
* Апельсин следует за ним с задержкой 1 сек
* Банан — ещё через 1 сек
Визуально:
* Невидимая "кисть" проходит по фруктам и перекрашивает
* При перекраске: небольшая вспышка яркости (0.2 сек)
* Цвет меняется плавно: 0.5 сек transition
Палитра волны: Красный → Оранжевый → Жёлтый → Зелёный → Голубой → Синий → Фиолетовый → Красный
Блок 2 (4–8 мин): ШАХМАТНЫЙ ПАТТЕРН — 6 ФРУКТОВ (3×2 сетка)
Стартовая позиция:
Яблоко(красн)    Морковь(оранж)   Банан(жёлт)
Помидор(красн)   Апельсин(оранж)  Ананас(жёлт)


Паттерн 1: Шахматная смена
* Чётные (яблоко, банан, апельсин) меняются на синий
* Нечётные (морковь, помидор, ананас) остаются
* Через 5 сек: меняются обратно
* Медленная пульсация красный↔синий
Паттерн 2: Волна по диагонали
* Волна идёт по диагонали (верхний-левый → нижний-правый)
* Задержка: каждый следующий на диагонали +0.8 сек
Блок 3 (8–12 мин): РАДУЖНАЯ ВОЛНА — 5 ФРУКТОВ В РЯД
5 фруктов в ряду. Каждый изначально своего естественного цвета. Радужная волна проходит слева направо — перекрашивает в порядке радуги. Скорость волны: 1 фрукт/сек.
Момент синхронизации (каждые 30 сек): Все 5 фруктов одновременно становятся одним цветом → пауза 3 сек → волна снова.
Блок 4 (12–16 мин): ОБРАТНАЯ ВОЛНА И ВСТРЕЧНАЯ
* Прямая волна идёт слева направо (красный→фиолетовый)
* Встречная волна идёт справа налево (фиолетовый→красный)
* Они встречаются в центральном фрукте
* Центральный фрукт: смесь обоих цветов → белый вспыхивает
Блок 5 (16–20 мин): СЛУЧАЙНАЯ СМЕНА (как пиксели)
Каждый фрукт меняет цвет независимо:
* Период смены: 3–7 сек (разный у каждого)
* Случайный следующий цвет из радуги
* Смена: плавная 0.5 сек transition
* Эффект: живое переливающееся полотно
Блок 6 (20–25 мин): МЕДЛЕННОЕ ЗАТУХАНИЕ
Все фрукты постепенно теряют насыщенность:
* 100% saturation → 60% → 30% → серые → постепенно назад
* Очень медленно — 5 мин на полный цикл
* Для засыпания
// Волна цвета
const colorWave = (fruitIndex: number, frame: number, waveSpeed: number) => {
  const wavePosition = (frame * waveSpeed) - fruitIndex * WAVE_GAP;
  const colorIndex = Math.floor(wavePosition / COLOR_DURATION) % COLORS.length;
  const progress = (wavePosition % COLOR_DURATION) / COLOR_DURATION;
  return lerpColor(COLORS[colorIndex], COLORS[(colorIndex + 1) % COLORS.length], progress);
};


// Плавная смена цвета при перекраске
const paintBrushEffect = (frame, changeFrame) => {
  const age = frame - changeFrame;
  if (age < 0) return 0;
  if (age < 6) return age / 6; // вспышка яркости (0.2 сек)
  return 1; // стабильный цвет
};


Музыка:
* BPM 72 — средний темп
* Каждый раз когда фрукт меняет цвет: тихий ксилофонный звук
* Разные ноты для разных цветов
* Когда много фруктов меняются → музыкальный аккорд
