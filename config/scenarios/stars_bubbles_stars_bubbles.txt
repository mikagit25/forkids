✨ Stars and Bubbles — Full Video Script
Duration: ~22 minutes | Age: 0–3 years | Language: English (minimal) | Format: Remotion Purpose: Pure entertainment — floating bubbles pop with satisfying sounds, stars twinkle and shoot across screen
________________


CORE DESIGN PRINCIPLES
* POP is the magic moment — every bubble pop is satisfying and rewarding
* Anticipation + release — bubble grows big… pauses… POP
* Stars are magical — trail light, twinkle, make wishes
* Contrast of movement — bubbles float slowly, stars shoot fast
* Sound design is 70% of the experience — soft POP, sparkle tinkle, whoosh
* Music: dreamy, gentle, 80 BPM — music box + soft synth

📋 REMOTION IMPLEMENTATION NOTES
Core components
* Bubble — circle with transparency, rainbow shimmer, wobble physics
* BubblePop — ring expand + sparkle burst at pop point
* Star — 5-point polygon, rotation, twinkle (scale pulse)
* ShootingStar — star + fading trail, bezier path motion
* SparkleParticle — tiny bright point, gravity + fade
* BubbleRise — bubble with gentle sway physics floating upward
* StarTrail — series of fading circles following star path
Bubble transparency / shimmer

const shimmerColor = `hsla(${hue}, 80%, 75%, 0.3)`;

Bubble wobble physics
const wobbleX = Math.sin(frame * 0.08) * wobbleAmount;
const wobbleY = Math.cos(frame * 0.06) * wobbleAmount * 0.7;
const scaleX = 1 + wobbleX * 0.05;
const scaleY = 1 + wobbleY * 0.05;


Bubble rise with sway
const riseSpeed = 1.5; // pixels per frame
const swayAmount = 20;
const swaySpeed = 0.03;
const x = startX + Math.sin(frame * swaySpeed) * swayAmount;
const y = startY - frame * riseSpeed;


Pop effect
// On pop frame: spawn 8-12 sparkle particles
const sparkles = Array.from({ length: 10 }).map((_, i) => {
  const angle = (i / 10) * Math.PI * 2;
  const speed = 3 + Math.random() * 4;
  return {
    vx: Math.cos(angle) * speed,
    vy: Math.sin(angle) * speed,
    life: 20 + Math.random() * 10,
  };
});
