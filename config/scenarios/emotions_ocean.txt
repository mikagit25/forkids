СЧАСТЬЕ (Happy / سعيد)
HAPPY-A — БЕЗ СЛОВ (25 мин)
BPM: 88 | Яркие тёплые цвета
Блок 1: КРУГЛИК ПРОСЫПАЕТСЯ СЧАСТЛИВЫМ
* Спит. Просыпается — глаза открываются → сразу ШИРОКАЯ УЛЫБКА.
* Прыгает от радости. BOB очень бодрый.
* Танцует сам с собой.
Блок 2: ПРИЧИНЫ СЧАСТЬЯ
* Мама приходит → Круглик счастлив (обнимает).
* Любимая еда → Круглик счастлив (глаза-сердечки).
* Солнечный день → Круглик счастлив (бежит к свету).
Блок 3: СЧАСТЬЕ ЗАРАЗНО
* Круглик встречает грустного маленького кружочка.
* Делится радостью — танцует рядом.
* Грустный кружочек начинает улыбаться → тоже прыгает.
Блок 4: СМЕХ
* Что-то смешное (не показываем что — персонаж смотрит вниз).
* Круглик смеётся — трясётся весь — сгибается.
* Слёзы от смеха (но весёлые).
Блок 5: СОННОЕ СЧАСТЬЕ
* День прошёл. Круглик устал но доволен.
* Ложится с улыбкой. Улыбка держится во сне.

ГРУСТЬ (Sad / حزين)
SAD-A — БЕЗ СЛОВ (25 мин)
BPM: 58 | Голубые/серые тона
Блок 1: КРУГЛИК ГРУСТИТ
* Уголки рта вниз. Глаза немного меньше.
* Медленный BOB — почти не прыгает.
* Голова чуть опущена.
Блок 2: СЛЁЗА
* Одна слеза медленно скатывается.
* Падает — маленький круг на полу.
* Круглик смотрит на слезу — снова грустит.
Блок 3: УТЕШЕНИЕ
* Появляется маленький кружочек — обнимает Круглика.
* Круглик обнимает в ответ.
* Медленно-медленно начинает улыбаться.
Блок 4: ЛУЧШЕ СТАНОВИТСЯ
* Улыбка растёт. Цвета экрана теплеют.
* К концу: снова BOB — небольшой но настоящий.
Важно: грусть показана как нормальное состояние. Не страшно. Проходит.

УДИВЛЕНИЕ (Surprised / مندهش)

* Маленькое становится огромным
* Невидимое становится видимым
Блок 3: ХОРОШИЕ СЮРПРИЗЫ
* Торт! Подарок! Друг! → удивление + счастье.

// Эмоции — морфинг лица
const interpolateEmotion = (from, to, progress) => ({
  eyeScale: lerp(from.eyeScale, to.eyeScale, progress),
  mouthCurve: lerp(from.mouthCurve, to.mouthCurve, progress),
  browAngle: lerp(from.browAngle, to.browAngle, progress),
  color: lerpColor(from.color, to.color, progress),
});


// Дыхательный шар — связь с сенсорными видео
const breathingBall = (frame) => {
  const cycle = (frame % 300) / 300;
  return cycle < 0.4
    ? interpolate(cycle, [0, 0.4], [minR, maxR], { easing: Easing.inOut(Easing.cubic) })
    : interpolate(cycle, [0.5, 0.9], [maxR, minR], { easing: Easing.inOut(Easing.cubic) });
};


// Кит — хвост вверх-вниз
const whaleTail = (frame) => {
  const tailAngle = Math.sin(frame * 0.04) * 20; // медленный мах
  const bodyY = Math.sin(frame * 0.04 + Math.PI * 0.5) * 8; // тело следует
  return { tailAngle, bodyY };
};


// Осьминог — 8 независимых щупалец
const tentacles = Array.from({length: 8}).map((_, i) => ({
  baseAngle: (i / 8) * 2 * Math.PI,
  phase: i * 0.4, // каждое своей фазой
  amplitude: 30 + rand(-10, 10),
  frequency: 0.06 + rand(-0.02, 0.02),
  getAngle: (frame) => Math.sin(frame * freq + phase) * amplitude,
}));

const brickLayer = {
  rows: [],
  addBrick: (frame, targetRow, targetCol) => {
    if (frame === brickTiming[targetRow][targetCol]) {
      rows[targetRow][targetCol] = { visible: true, dropProgress: 0 };
    }
    // Кирпич падает сверху на место: y: -20 → finalY
    const dropProgress = interpolate(frame, [dropStart, dropStart + 20], [0, 1], {
      easing: Easing.out(Easing.bounce),
      extrapolateRight: 'clamp',
    });
  },
};
