📋 REMOTION IMPLEMENTATION NOTES
Components needed
* FlowerCharacter — animated flower with face, various species variants
* GardenBackground — layered garden: path, flower beds, wall, sky
* WateringCan — watering action with water stream physics
* WaterStream — arc of water falling from can to flowers
* ButterflyCharacter — butterfly with wing flap animation
* FireflyGlow — pulsing point light with organic movement
* FlowerBloom — petal opening animation from bud to full bloom
* PetalFall — petals drifting downward with slight rotation
* GardenPond — reflective surface showing mirror image
Flower bloom animation
const bloomProgress = interpolate(
  frame,
  [bloomStart, bloomStart + 30],
  [0, 1],
  { extrapolateRight: 'clamp', easing: Easing.out(Easing.back(1.2)) }
);
// Each petal rotates from closed (pointing up) to open (pointing out)
const petalAngle = interpolate(bloomProgress, [0, 1], [-70, 0]);
const petalScale = interpolate(bloomProgress, [0, 1], [0.1, 1]);


Watering can stream
// Parabolic arc from spout to ground
const streamPoints = Array.from({ length: 20 }).map((_, i) => {
  const t = i / 20;
  return {
    x: spoutX + t * horizontalDist,
    y: spoutY + t * verticalDist - t * (1 - t) * arcHeight,
    opacity: 1 - t * 0.3,
    size: 4 + t * 3,
  };
});


Butterfly wing flap
const flapSpeed = 0.25; // rotations per frame
const wingAngle = Math.sin(frame * flapSpeed) * 35; // degrees
// Left wing: +wingAngle, Right wing: -wingAngle (mirror)
// Also slight body bob synchronized
const bodyY = Math.abs(Math.sin(frame * flapSpeed)) * 3;


Firefly glow
// Organic pulsing — not perfectly regular
const glowPhase = frame * 0.05 + fireflyIndex * 1.3;
const glowIntensity = Math.sin(glowPhase) * 0.4 + 0.6;
const glowRadius = (Math.sin(glowPhase) * 0.3 + 0.7) * maxRadius;
// Organic drift movement
const driftX = Math.sin(frame * 0.02 + fireflyIndex) * 30;
const driftY = Math.cos(frame * 0.015 + fireflyIndex * 1.7) * 20;


Garden reflections in pond
// Mirror all elements above pond line

Audio design
* Song BPM = 90 — gentle, flowing like garden breeze
* Water pour: continuous soft stream
* Flower bloom: soft pop sound like opening
* Butterfly: soft wing flutter (barely audible)
* Bee: gentle BZZZ — friendly, not scary
* Firefly: soft electric blink sound
* Rain: gentle consistent soft rain loop
* Garden ambience: birds, insects, breeze — layered
* Night garden: crickets, owl, silence
* Final kecapi note: distinctly Indonesian
Special Indonesian cultural elements
* Melati (jasmine) is Indonesia's national flower — give it extra love
* Mawar (rose) is universally beloved — classic beauty
* Show Indonesian garden aesthetic — lush tropical, layered
* Warm colors of Indonesian sunrise/sunset
* Natural materials: bamboo fence, terra cotta pots
* Make it feel authentically Indonesian — not generic
