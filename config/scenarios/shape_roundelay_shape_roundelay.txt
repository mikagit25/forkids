📋 REMOTION IMPLEMENTATION NOTES
Core components
* SpinningShape — any shape, rotation animation, speed prop
* OrbitingShape — shape following circular path, radius + speed props
* ShapeMorph — SVG path interpolation between circle/square/triangle/star
* Roundelay — N shapes in circle formation, rotating as group
* MandalaBloom — radial symmetry, concentric rings appearing over time
* FireworkBurst — shape explodes into N copies flying outward with trails
* ColorWave — color ripples around a ring of shapes sequentially
Shape morphing (SVG path interpolation)

Roundelay formation
const shapes = 6;
const radius = 200;
shapes.map((_, i) => {
  const angle = (i / shapes) * Math.PI * 2 + (frame / 60) * rotationSpeed;
  const x = centerX + Math.cos(angle) * radius;
  const y = centerY + Math.sin(angle) * radius;
  return { x, y };
});


Mandala (radial symmetry)
const symmetry = 8;
Array.from({ length: symmetry }).map((_, i) => {
  const angle = (i / symmetry) * Math.PI * 2;
  return (
    <g transform={`rotate(${angle * (180/Math.PI)} ${cx} ${cy})`}>
      {renderShapeRing(frame)}
    </g>
  );
});
