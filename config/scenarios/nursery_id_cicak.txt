📋 REMOTION IMPLEMENTATION NOTES
Components needed
* CicakCharacter — gecko body: head, 4 legs with sticky pads, tail, big eyes
* CicakCrawl — movement along any surface (wall, ceiling, floor)
* Tongue — rapid extend/retract animation for HAP moment
* Mosquito — flying insect with zigzag path, buzzing wings
* WallSurface — textured wall background with depth
* StickyFootprint — small footprint that appears and fades
* HAPEffect — flash + tongue snap + catch particle burst
* CeilingMode — flip all characters upside down naturally
Cicak movement on wall
// Cicak can be on any surface — rotate and position accordingly
const surfaces = {
  wall:    { rotation: 0,   gravityDir: 'down' },
  ceiling: { rotation: 180, gravityDir: 'up'   },
  floor:   { rotation: 0,   gravityDir: 'down' },
};


// Sticky creeping movement
const legPositions = [
  { x: -20, y: -15, phase: 0   }, // front left
  { x:  20, y: -15, phase: 0.5 }, // front right
  { x: -20, y:  15, phase: 0.5 }, // back left
  { x:  20, y:  15, phase: 0   }, // back right
];


legPositions.map(leg => {
  const step = Math.sin((frame * walkSpeed) + leg.phase * Math.PI) > 0;
  const legY = step ? leg.y - 8 : leg.y; // lift leg when stepping
});


HAP tongue animation
const hapFrame = triggerFrame;
const tongueProgress = interpolate(
  frame,
  [hapFrame, hapFrame + 4, hapFrame + 8],
  [0, 1, 0],
  { extrapolateRight: 'clamp', easing: Easing.out(Easing.exp) }
);
// Tongue extends to mosquito position then retracts
const tongueLength = tongueProgress * distanceToMosquito;
const tongueAngle = angleTo(cicakPos, mosquitoPos);


Mosquito zigzag flight
const mosquitoX = startX + frame * 2 + Math.sin(frame * 0.4) * 40;
const mosquitoY = startY + Math.cos(frame * 0.3) * 30;
// Wing flap
const wingAngle = Math.sin(frame * 0.8) * 30;


Eyes tracking mosquito
// Cicak eyes rotate to follow mosquito position
const eyeAngle = Math.atan2(
  mosquitoY - cicakY,
  mosquitoX - cicakX
);
// Pupils offset in eye direction
const pupilX = Math.cos(eyeAngle) * 5;
const pupilY = Math.sin(eyeAngle) * 5;

Audio design
* Song BPM = 100 → 1 beat = 18 frames
* "Cit cit" cicak call: soft, high-pitched, every 8 beats during intro
* Mosquito BZZZ: continuous, gets louder as it approaches
* Pre-HAP silence: exactly 2 seconds (60 frames)
* HAP sound: sharp, punchy, satisfying click/snap
* Tongue snap: rapid whoosh — in and out in 0.25 seconds
* Victory fanfare: short 3-note ascending brass sting
Key moment: HAP timing
The HAP must be perfectly satisfying — this is the whole song's payoff.
* Build silence: 45 frames
* HAP visual: frame 0
* Tongue fully extended: frame 4
* Mosquito captured: frame 5
* Tongue retracting: frames 5–8
* Cicak swallowing: frame 10
* Victory dance begins: frame 15
