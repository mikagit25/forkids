ВИДЕО 3.1 — ПАДЕНИЕ И ОТСКОК
Duration: 25 min | Loop: 3 min × 8 | BPM: 80
Концепция
Фрукты падают с разной высоты и по-разному отскакивают. Физика bounce — самый satisfying визуальный паттерн. Каждый фрукт имеет свой характер упругости.
Параметры упругости по фруктам:
Фрукт
	Упругость
	Поведение
	Яблоко
	0.75
	4–5 отскоков, уменьшаются
	Арбуз
	0.40
	1–2 отскока, тяжёлый
	Клубника
	0.85
	6–7 отскоков, лёгкая
	Банан
	0.50
	3 отскока + скользит
	Апельсин
	0.70
	4 отскока
	Вишня
	0.90
	8+ отскоков, очень лёгкая
	Структура блоков:
Блок 1 (0–3 мин): ОДНО ЯБЛОКО С РАЗНОЙ ВЫСОТЫ
* 10 сек: яблоко падает с высоты 100px → отскакивает 3 раза → стоп
* 15 сек: падает с 400px → 5 отскоков → стоп
* 20 сек: падает с полной высоты (1080px) → 6 отскоков → стоп
* 10 сек: пауза — яблоко стоит на земле
Блок 2 (3–6 мин): ТРИ ФРУКТА ОДНОВРЕМЕННО
* Яблоко (слева), апельсин (центр), клубника (справа)
* Все падают с одной высоты — разные отскоки
* Яблоко: 4 отскока, апельсин: 4, клубника: 6+
* Последняя клубника всё ещё прыгает когда яблоко давно стоит — смешно!
Блок 3 (6–9 мин): КАСКАД — ОДИН ЗА ДРУГИМ
* Яблоко падает → ещё не остановилось → падает апельсин → потом клубника
* 5 сек между падениями
* Все прыгают в разных фазах → живая "аранжировка"
Блок 4 (9–12 мин): АРБУЗ — КОНТРАСТ
* Один огромный арбуз падает с высоты
* 1 большой отскок — очень медленный (тяжёлый!)
* Потом маленькая вишня — 10+ отскоков — быстрые-быстрые
* Контраст масс: тяжёлое и лёгкое
Блок 5 (12–15 мин): БАНАН СКОЛЬЗИТ
* Банан падает горизонтально (плашмя)
* Приземляется → отскакивает → скользит по полу
* При скольжении: скорость затухает — трение
* Врезается в края экрана — отскакивает от стенок
Блок 6 (15–18 мин): ДОЖДЬ ИЗ ФРУКТОВ
* 8–10 разных фруктов падают с разных позиций
* Все одновременно — как дождь
* Каждый со своей упругостью
* Экран внизу: ковёр из подпрыгивающих фруктов
Блок 7 (18–21 мин): ЗАМЕДЛЕННЫЙ ОТСКОК
* Всё в 3× медленнее
* Видно каждую деформацию при приземлении
* squash в момент касания очень заметен
* Красиво и неожиданно в слоумо
Блок 8 (21–25 мин): ЗАТИХАНИЕ
* Фрукты падают и отскакивают всё меньше и меньше
* Постепенно все стоят
* Последний: вишня — долго-долго прыгает крошечными прыжками
// Физика отскока
class BounceFruit {
  y: number;
  vy: number = 0;
  readonly restitution: number; // упругость 0-1


  update() {
    this.vy += GRAVITY; // 0.8 px/frame²
    this.y += this.vy;


    if (this.y >= GROUND_Y) {
      this.y = GROUND_Y;
      this.vy = -Math.abs(this.vy) * this.restitution;
      // Squash на приземлении
      this.triggerSquash();
    }
  }


  triggerSquash() {
    // scaleX 1→1.4→1 за 8 кадров
    // scaleY 1→0.6→1 за 8 кадров
    this.squashProgress = 0;
  }
}


// Squash анимация
const squashX = interpolate(squashProgress, [0, 4, 8], [1, 1.4, 1]);
const squashY = interpolate(squashProgress, [0, 4, 8], [1, 0.6, 1]);

ВИДЕО 3.2 — МАГНИТЫ-ФИГУРЫ
Duration: 25 min | Loop: 4 min × 6 | BPM: 65
Концепция
Фигуры притягиваются и отталкиваются как магниты. Предсказуемое напряжение — ребёнок ждёт момента касания. Момент "почти касания" — 2-секундная пауза — самый напряжённый.
Детальный цикл притяжения (60 сек):
Фаза 1 — РАССТОЯНИЕ (0–15 сек)
* Два круга на противоположных сторонах экрана
* Медленно движутся друг к другу
* Скорость: constant 2px/сек
Фаза 2 — ЗОНА ПРИТЯЖЕНИЯ (15–30 сек)
* Расстояние < 200px: ускорение (магнитное притяжение)
* velocity += attraction_force / distance²
* Скорость растёт: 2 → 4 → 8px/сек
Фаза 3 — НАПРЯЖЕНИЕ (30–32 сек)
* Расстояние < 20px: полная остановка — 2 сек тишины
* Оба круга вибрируют на месте — дрожь
* Музыка замирает
Фаза 4 — КАСАНИЕ (32 сек)
* КОНТАКТ → вспышка
* Оба деформируются в точке касания
* squash в стороны
Фаза 5 — ОТТАЛКИВАНИЕ (32–45 сек)
* Моментальное отталкивание — обратная скорость
* Улетают в стороны — ускоряются
* Достигают края — мягко отскакивают от стенки
* Замедляются в центре
Фаза 6 — РАВНОВЕСИЕ (45–60 сек)
* Обе фигуры нашли равновесие
* Медленно покачиваются притягиваясь-отталкиваясь
* Не касаются — но почти
Вариации по блокам:
Блок 1: ДВА КРУГА (одинаковые)
Классическое притяжение.
Блок 2: БОЛЬШОЙ И МАЛЕНЬКИЙ
Маленький круг летит к большому быстрее. Большой почти не двигается — как звезда и планета.
Блок 3: ТРИ В ТРЕУГОЛЬНИКЕ
Три круга — все притягиваются друг к другу. Находят равновесие в равностороннем треугольнике. Если нарушить — долго ищут новое равновесие.
Блок 4: ОДИН ПРОТИВ ДВУХ
Один большой отталкивает двух маленьких. Маленькие пытаются обойти и притянуться.
Блок 5: РАЗНЫЕ ЗНАКИ
Красный (N) и синий (S) — притягиваются. Два красных — отталкиваются. Три фигуры: RRS — интересная динамика.
Блок 6: МЕДЛЕННЫЙ ТАНЕЦ
Пара кружится вокруг общего центра масс. Как двойная звезда. 30 сек/оборот.
// Физика магнитов
const ATTRACTION_CONST = 5000; // настраиваем для красивого поведения
const REPULSION_DIST = 80; // px — расстояние отталкивания


const updateMagnet = (a, b) => {
  const dx = b.x - a.x;
  const dy = b.y - a.y;
  const dist = Math.sqrt(dx*dx + dy*dy);


  const forceMag = ATTRACTION_CONST / (dist * dist);
  const forceSign = dist < REPULSION_DIST ? -1 : 1; // притяжение или отталкивание


  const fx = (dx / dist) * forceMag * forceSign;
  const fy = (dy / dist) * forceMag * forceSign;


  a.vx += fx / a.mass;
  a.vy += fy / a.mass;
  // Damping (затухание)
  a.vx *= 0.98;
  a.vy *= 0.98;
};

ВИДЕО 3.3 — ФРУКТЫ ПОД ВОДОЙ
Duration: 30 min | Loop: 5 min × 6 | BPM: 55
Концепция
Подводная сцена. Фрукты медленно покачиваются в воде. Пузырьки поднимаются. Свет преломляется сверху. Самое медитативное видео всего проекта.
Фон:
* Глубокий синий-бирюзовый (#0A2A3D)
* Снизу: более тёмный (#061825)
* Лучи света сверху: 5–7 диагональных светлых полос (очень прозрачных)
* Лучи медленно покачиваются (sin-волна 20 сек/цикл)
Фрукты под водой — движение:
Базовое движение (для всех):
// Подводное плавание — более плавное чем воздух
const underwaterX = Math.sin(frame * 0.015 + phaseX) * amplitudeX;
const underwaterY = Math.sin(frame * 0.010 + phaseY) * amplitudeY;
// Частоты медленнее чем обычные движения


Яблоко под водой:
* Не тонет (яблоко легче воды — реально!)
* Находится в верхней трети экрана
* Медленно покачивается: 3 сек/цикл
Банан под водой:
* Немного тонет (банан тяжелее воды)
* В нижней трети экрана
* При подъёме — изгибается вперёд; при опускании — назад
Арбуз:
* Тяжёлый — у самого дна
* Еле движется: амплитуда 10px
* Но пузырьки вокруг него активные
Виноград:
* Гроздь плавает как единое целое
* Ягодки немного разлетаются при движении
* Собираются обратно
Пузырьки:
// Пузырьки поднимаются от дна
const bubble = {
  x: randomX,
  y: screenH,
  size: random(5, 20),
  riseSpeed: random(0.5, 2), // px/frame
  wobble: random(0.02, 0.06), // горизонтальное покачивание
};


// Обновление
bubble.y -= bubble.riseSpeed;
bubble.x += Math.sin(frame * bubble.wobble) * 2;
if (bubble.y < 0) respawnBubble(bubble);


Световые лучи:
// Диагональные полосы света
const rays = Array.from({length: 6}).map((_, i) => ({
  x: i * 200,
  angle: -30, // градусов
  width: 60,
  opacity: 0.04 + Math.sin(frame * 0.008 + i) * 0.02,
}));


Структура блоков (30 мин):
* Блок 1: 3 фрукта — яблоко, клубника, апельсин (маленький набор)
* Блок 2: + добавляется банан
* Блок 3: + добавляется арбуз (большой, у дна)
* Блок 4: 5 фруктов — все плавают
* Блок 5: фрукты медленно уплывают за края → экран с только пузырьками
* Блок 6: ночная версия — всё темнее, меньше света, медленнее
Музыка:
* Подводные ambient звуки (низкий гул)
* Пузырьки: тихий "bloop" при рождении
* Медленная флейта — 55 BPM
* Эффект задержки (reverb) — как под водой

ВИДЕО 3.4 — ТЕНИ ФИГУР
Duration: 25 min | Loop: 4 min × 6 | BPM: 70
Концепция
Светлый фон. Фигуры движутся и отбрасывают тени. Источник света движется → тени поворачиваются и меняют размер. Совершенно новый визуальный опыт для ребёнка.
Фон:
* Светлый кремовый (#F5F0E8)
* Имитация поверхности стола или пола
Механика теней:
// Источник света двигается по кругу
const lightAngle = (frame / (lightPeriod * 30)) * 2 * Math.PI;
const lightX = cx + Math.cos(lightAngle) * lightRadius;
const lightY = cy - 300; // всегда сверху


// Тень = проекция фигуры от источника света на пол
const shadowScale = groundY / (groundY - lightY + object.y);
const shadowX = object.x + (object.x - lightX) * (groundY - object.y) / (lightY - object.y);
const shadowOffsetX = shadowX - object.x;
const shadowOpacity = interpolate(Math.abs(lightX - object.x), [0, 400], [0.7, 0.2]);


Структура блоков:
Блок 1: ОДИН КРУГ — ВРАЩАЮЩИЙСЯ СВЕТ
* Один синий круг в центре
* Источник света медленно облетает его по кругу (20 сек/оборот)
* Тень поворачивается и вытягивается
Блок 2: ТРИ ФИГУРЫ — НЕПОДВИЖНЫЙ СВЕТ
* Три фигуры в ряд: круг, квадрат, треугольник
* Источник света стоит — тени фиксированные
* Фигуры начинают двигаться (BOB) — тени тоже двигаются
* Тень квадрата и треугольника интереснее тени круга!
Блок 3: ФРУКТЫ — ДВИЖУЩИЙСЯ СВЕТ СЛЕВА НАПРАВО
* Яблоко, банан, морковь на "полке"
* Свет движется слева направо за 30 сек
* Тени поворачиваются: смотрят вправо → прямо → влево
Блок 4: ДВА ИСТОЧНИКА СВЕТА
* Два источника: красный (левый) и синий (правый)
* Каждый создаёт тень своего цвета
* Две тени: красная (вправо) и синяя (влево)
* В зоне перекрытия теней: фиолетовая тень!
Блок 5: ОЧЕНЬ КОРОТКАЯ ТЕНЬ / ОЧЕНЬ ДЛИННАЯ
* Свет высоко над объектом: тень маленькая
* Свет опускается к горизонту: тень вытягивается
* Имитация утра/вечера
Блок 6: НЕСКОЛЬКО ИСТОЧНИКОВ — ХАОС ТЕНЕЙ
* 4 источника света расставлены по экрану
* Каждый создаёт тень
* Сложный узор теней вокруг фигур
Музыка:
* BPM 70
* Тихая ambient музыка — как полдень
* При движении тени: лёгкий шелест
* Без ударных — спокойно

  x: obj.x + (obj.x - light.x) * (ground - obj.y) / (light.y - ground),
