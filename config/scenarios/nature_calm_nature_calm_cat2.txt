🐠 СЦЕНАРИЙ 2.1 — UNDERWATER WORLD
Duration: 22 min | Age: 0–2 years
Concept
Peaceful underwater scene — fish swim slowly — no sudden movements. One of the most calming visual experiences for babies. Scientifically proven: aquarium watching reduces stress in infants. Parents love it: no overstimulation, beautiful visuals.
Core design rules:
* Everything moves SLOWLY — max speed: gentle drift
* Colors: soft blues, teals, coral pinks — never harsh
* No predators — no violence — only peaceful coexistence
* Music: ambient underwater, soft bubbling, no drums
* BPM: 60 maximum

Butterfly wing cycle:
// Wing opens and closes — Monarch: 0.6 sec cycle
const wingCycle = (frame % 18) / 18; // 18 frames = 0.6 sec
const wingAngle = Math.cos(wingCycle * Math.PI * 2) * 45; // 45 degree max
// Left wing: +wingAngle, Right wing: -wingAngle
// Body moves UP slightly when wings close
const bodyY = -Math.abs(Math.cos(wingCycle * Math.PI * 2)) * 3;
