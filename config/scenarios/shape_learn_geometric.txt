⭕ ГЕОМЕТРИЧЕСКИЕ ФИГУРЫ — Серия 30-минутных видео
Метод: One Concept Deep | Три измерения: Форма + Цвет + Количество Серия: 8 фигур × 30 минут = 4 часа контента
________________


ГЛАВНАЯ ИДЕЯ СЕРИИ
Проблема обычных видео о фигурах: показывают круг, квадрат, треугольник в одном видео — дети путают.
Наш подход: одна фигура на 30 минут, но с тремя измерениями:
1. ФОРМА — что это, как называется, где встречается
2. ЦВЕТ — та же фигура в разных цветах (форма не меняется — только цвет)
3. КОЛИЧЕСТВО — 1, 2, 3 фигуры — счёт через знакомую форму
Ключевой инсайт блока E (цвет меняется): ребёнок видит КАК форма остаётся кругом когда цвет меняется с красного на синий. Это строит понимание что форма и цвет — разные свойства одного предмета.
________________

БЛОК A — ЗНАКОМСТВО С КРУГОМ (0:00–2:00)
Animation
* Белый экран — полная тишина 1 сек
* Из центра медленно расплывается синий круг
* Растёт... останавливается в центре — идеальный круг
* Шейпи (персонаж-круг) появляется в середине: глаза, улыбка
* Рука рисует круг вокруг него одной линией: "Вот — CIRCLE!"
* Текст: CIRCLE — появляется под фигурой
Narrator (медленно, чётко):
"Circle." (пауза 2 сек) "Cir-cle." (пауза 2 сек) "A circle is round." (пауза 3 сек) "No corners. Round." (пауза 2 сек) "Circle."
Ключевое слово этого видео: ROUND (круглый)
Каждый раз при показе круга произносится "round".

БЛОК B — КРУГ В МИРЕ (2:00–5:00)
6 примеров × 30 сек = 3 мин Каждый: предмет появляется → стрелка по контуру → "Circle! Round!"
Солнце: "The SUN is a circle! Round!" Луна: "The MOON is a circle! Round!" Мяч: "A BALL is a circle! Round!" Колесо: "A WHEEL is a circle! Round!" Тарелка: "A PLATE is a circle! Round!" Монета: "A COIN is a circle! Round!"
После каждого — маленький круг в коллекции справа накапливается. Финал: "Everything round is a CIRCLE!"

БЛОК C — ДИАЛОГ: ФОРМА (5:00–7:00)
4 раунда × 30 сек
Раунд 1: Большой синий круг на экране.
"Is this a circle?" (пауза 4 сек) "YES! Circle! Round!"
Раунд 2: Квадрат появляется (контраст).
"Is this a circle?" (пауза 4 сек) "No! A square has corners!" (сразу — круг появляется рядом) "But THIS is a circle! Round — no corners!"
Раунд 3: Три фигуры — круг, звезда, треугольник.
"Can you find the circle?" (пауза 4 сек) Стрелка на круг → "There! CIRCLE!"
Раунд 4: Шейпи спрашивает.
"What shape am I?" (пауза 5 сек) Шейпи указывает на себя → "CIRCLE! I'm a circle!"

БЛОК E — ЦВЕТ МЕНЯЕТСЯ (9:00–11:00) 🌟 ГЛАВНЫЙ БЛОК
Концепция
Стоит ОДИН круг. Медленно меняет цвет. Ребёнок видит: форма — ОДНА И ТА ЖЕ. Только цвет меняется. Это главный инсайт всего видео.
Animation
* Один большой круг в центре — красный
* Медленно (3 сек) переходит в → синий
* Медленно → жёлтый
* Медленно → зелёный
* Медленно → фиолетовый
* Медленно → оранжевый
* Назад → красный
При каждой смене:
* Narrator называет новый цвет
* Текст меняется: RED → BLUE → YELLOW...
* Шейпи выглядит одинаково — только цвет другой
Narrator:
"Look! One circle." (пауза 1 сек) "RED circle." (цвет начинает меняться) "Now... BLUE! Blue circle!" (пауза 2 сек) "Same circle! Different color!" (пауза 2 сек) "Now... YELLOW! Yellow circle!" (пауза 2 сек) "The circle doesn't change. Only the COLOR changes!" (пауза 3 сек) "GREEN circle! Still a circle! Just green now!"
Ключевая фраза которая повторяется каждый раз:
"Same circle! Different color!"

БЛОК H — СЧИТАЕМ 1-2-3 (15:00–18:00)
Animation — медленный satisfying счёт
* Пустой экран
* Первый круг падает сверху — ПЛЮХ — 1 появляется над ним

БЛОК J — ВСЁ ВМЕСТЕ (20:00–23:00)
Три измерения в одной сцене
Animation
* 3 синих круга стоят в ряд
* Arrows указывают на: → ФОРМА: "Circle! Round!" → ЦВЕТ: "Blue!" → КОЛИЧЕСТВО: "Three!"

Последняя вариация — важная: Три синих квадрата.
"Three blue... SQUARES! Not circles — squares have corners!" Возврат к кругам: "Our circle has NO corners. Round!"

БЛОК L — ПЕТЛЯ-ГИМН (26:00–30:00)
Концепция
Satisfying-луп: несколько кругов стоят в ряду и медленно синхронно меняют цвет. Монотонно, красиво, гипнотично — для закрепления и засыпания.
Animation
* 5 кругов разного размера стоят в ряду
* Все ОДНОВРЕМЕННО меняют цвет плавно: красный → синий → жёлтый → зелёный → фиолетовый
* Цикл 8 секунд — бесконечный
* Очень мягко, никаких прыжков
Narrator (очень тихо, редко):
"Circles... changing colors... always circles..." (каждые 15 сек)
Это и завершение видео И промо-механика:
Дети хотят смотреть снова — потому что луп красивый и успокаивает.

🟥 ВИДЕО 2 — SQUARE (Квадрат)
Duration: 30 min
Ключевые отличия от видео про круг:
Блок A:
* Рука рисует квадрат — 4 отдельные линии — каждая останавливается на углу
* "One corner... two corners... three corners... FOUR corners!"
* Ключевые слова: CORNERS (углы), FOUR SIDES
Блок B — Квадрат в мире:
Окно, крекер, книга (обложка), кубик, тарелка-квадрат, пол-плитка
Блок C — Сравнение с кругом (новый элемент!):
"Remember the circle? Round — no corners." "The square has FOUR corners!" (считаем вместе: 1, 2, 3, 4) Круг и квадрат рядом — ребёнок видит разницу.
Блок E — Цвет меняется:
Та же механика — один квадрат меняет цвет. Акцент: "Still a square! Just a different color!"
Блок H — Считаем:
1 квадрат → 2 → 3 → 4 (четыре = число углов! Связь!)
"Four squares! Just like four corners!"
Песенка:
🎵 "Square, square — FOUR corners! Four sides, four corners — square! RED square, BLUE square, GREEN! Best square I have ever seen! ONE square, TWO squares, THREE, FOUR! Squares, squares, let's count more!" 🎵

🔺 ВИДЕО 3 — TRIANGLE (Треугольник)
Duration: 30 min
Ключевые отличия:
Блок A:
* "Triangle has THREE corners!"
* Считаем: "One corner, two corners, THREE!"
* Ключевая связь: треугольник = TRI = THREE (три)
Блок B — Треугольник в мире:
Крыша дома, ломтик пиццы, дорожный знак, парусник (парус), горная вершина, кусок торта
Блок C — Тройное сравнение:
* Круг: 0 углов — ROUND
* Квадрат: 4 угла
* Треугольник: 3 угла Все три рядом — ребёнок строит систему.
Блок H — Считаем + связь с углами:
"Three triangles! Triangle has THREE corners — THREE triangles!" Визуально: три треугольника выстраиваются в бо́льший треугольник.
Блок L — Петля:
Треугольники вращаются медленно + меняют цвет одновременно.

ШАБЛОН ДИАЛОГОВЫХ БЛОКОВ (все видео)

Пауза 4 сек → "YES! [Shape]!"

Тип 3 (найди): "Can you find the [shape]?" (3 фигуры на экране)
Пауза 4 сек → стрелка → "There! [Shape]!"


Диалог: Цвет
Тип 1: "What color is the [shape]?"
Пауза 5 сек → персонаж "слышит" → "[COLOR]!"


Тип 2: "Is this a red or blue [shape]?"
Пауза 4 сек → правильный цвет светится → "[COLOR]!"


Тип 3: "Find the [color] [shape]!"
Пауза 4 сек → стрелка → "There it is!"


Диалог: Количество
Тип 1: "How many [shapes]?"
Пауза 4 сек → счёт вместе → "[NUMBER]!"


Тип 2: Шейпи показывает пальцы
"Show me with fingers! How many?"
Пауза 5 сек → шейпи поднимает пальцы → "[NUMBER]!"


Тип 3: Счёт с появлением
Фигуры появляются одна за одной → ребёнок считает → финальное число

ПЕТЛЯ-ГИМН (БЛОК L) — МЕХАНИКА ДЛЯ REMOTION
Это самый важный блок для watch time — дети смотрят снова и снова.
Базовая версия (все видео):
* N фигур стоят в ряду (3-7 в зависимости от фигуры)
* Все одновременно плавно меняют цвет
* Цикл: красный → оранжевый → жёлтый → зелёный → синий → фиолетовый → красный
* Длительность одного цвета: 3 секунды
* Переход: 2 секунды (плавный interpolate)
* Общий цикл: 30 сек × луп = 4 мин
Расширенная версия (последние 2 мин):
* Фигуры начинают двигаться — медленно покачиваться
* Разные фигуры двигаются в разном ритме (но все меняют цвет синхронно)
* Очень гипнотично
// ColorLoop component
const COLORS = ['#FF2222','#FF8800','#FFDD00','#22CC44','#2255FF','#8833FF'];
const cycleSeconds = 5; // сек на один цвет
const cycleFrames = cycleSeconds * 30;


const colorIndex = Math.floor((frame / cycleFrames) % COLORS.length);
const nextColorIndex = (colorIndex + 1) % COLORS.length;
const progress = (frame % cycleFrames) / cycleFrames;

// interpolateColor между двумя hex значениями
function lerpColor(a: string, b: string, t: number): string {
  const ah = parseInt(a.slice(1), 16);
  const bh = parseInt(b.slice(1), 16);
  const ar = (ah >> 16) & 0xff, ag = (ah >> 8) & 0xff, ab = ah & 0xff;
  const br = (bh >> 16) & 0xff, bg = (bh >> 8) & 0xff, bb = bh & 0xff;
  const r = Math.round(ar + (br - ar) * t);
  const g = Math.round(ag + (bg - ag) * t);
  const b2 = Math.round(ab + (bb - ab) * t);
  return `#${r.toString(16).padStart(2,'0')}${g.toString(16).padStart(2,'0')}${b2.toString(16).padStart(2,'0')}`;
}
