🌅 СЦЕНАРИЙ 4.1 — GOOD MORNING!
Duration: 20 min | Age: 1–3 years
Concept

🤝 СЦЕНАРИЙ 4.2 — FRIENDSHIP
Duration: 20 min | Age: 1–3 years
Concept
Two different animals meet — don't understand each other at first — become best friends. NO WORDS — told entirely through actions and expressions. Universal language — works in any culture. Teaches: kindness, sharing, helping, accepting differences.
Characters:
* BUMI: small round brown bear — shy, gentle
* KIKI: small round yellow bird — curious, energetic
Why they're different (the story engine):
* Bumi walks; Kiki flies
* Bumi eats berries; Kiki eats seeds
* Bumi is slow and careful; Kiki is fast and spontaneous
* Bumi is big (relatively); Kiki is tiny
* These differences CREATE their friendship — they help each other

* Kiki shivers — Bumi curls around them — warm

* Coming so close but missing each other
* Music: gentle tension — not scary
The solution:
* Bumi has an idea — picks up stick — BANG on hollow log
* BOOM BOOM BOOM — sound carries far
* Kiki hears it — flies toward sound
* Bumi keeps banging
* Kiki appears through the fog
* Both stop — see each other

* Kiki chirps something
* Bumi grunts in agreement

* They understand each other
The end:

* Bumi slow — Kiki hops around Bumi
* Sometimes Kiki lands on shoulder
* Sometimes flies ahead then waits
* They've figured out how to be together
* Both their different speeds are okay
* Narrator (for the first time): "Different can be wonderful. That's what friendship is."

Concept

* Gets PROGRESSIVELY SLOWER and QUIETER
* Start: gentle but active — 85 BPM
* End: nearly silent — 50 BPM
* Light dims gradually throughout
* Music instruments drop out one by one
* Last 5 minutes: just lullaby humming

REMOTION NOTES — CATEGORY 4
Core components:
   * CharacterBear / CharacterHedgehog / CharacterBird — main characters
   * SleepState — character with closed eyes, ZZZ bubble, breathing animation
   * DimmingOverlay — gradually darkening screen over time
   * WarmLighting — golden glow for indoor/morning scenes
   * EmotionExpression — face expression states: happy/worried/proud/loving
   * HugAnimation — two characters coming together
Progressive darkness for Goodnight video:
// Screen dims gradually from minute 0 to minute 20
const brightnessProgress = interpolate(frame, [0, totalFrames], [1.0, 0.4]);
// Apply CSS filter: brightness(brightnessProgress)
// Also reduce music volume: 1.0 → 0.2


Breathing sleep animation:
// Sleeping character's chest rises and falls
const sleepCycle = (frame % 90) / 90; // 3 second breath cycle
const chestRise = Math.sin(sleepCycle * Math.PI * 2) * 4; // 4px rise
// Also: occasional twitch (random small body movement every ~150 frames)


ZZZ bubble float:
const zzzBubbles = [
  { delay: 0, size: 16, path: 'gentle left arc' },
  { delay: 60, size: 20, path: 'gentle right arc' },
  { delay: 120, size: 24, path: 'gentle left arc' },
];
zzzBubbles.map(bubble => {
  const age = (frame - bubble.delay);
  if (age < 0) return null;
  const floatY = -age * 0.5; // float upward
  const opacity = interpolate(age, [0, 60, 120], [0, 0.9, 0]);
});
