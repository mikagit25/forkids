ЖИВОТНОЕ 1 — КОШКА (CAT)
ВИДЕО CAT-A — КОШКА СОЛО (25 мин, без слов)
BPM: 68 | Mood: грациозное, спокойное
Блок 1 (0–5 мин): КОШКА ПОТЯГИВАЕТСЯ
Появление: Кошка медленно выходит из-за правого края экрана. Видна сначала только мордочка — смотрит. Потом вся. Садится. Смотрит на камеру. Моргает медленно (Cat's slow blink = доверие).
STRETCH — растяжка: Кошка тянется вперёд: передние лапы выдвигаются, спина выгибается. Зад остаётся, хвост поднимается вертикально. 8 секунд. Медленно. Потом выпрямляется.
ARCH — выгибание спины: Спина выгибается вверх (испуганная поза но медленная = просто потягивается). Потом распрямляется. Повторяется 3 раза.
// Анимация выгиба спины кошки
// Кошка состоит из 5 сегментов (голова, шея, тело1, тело2, хвост)
const spineSegments = [head, neck, body1, body2, tail];
// При stretch: body1 и body2 опускаются, голова и хвост поднимаются
const archProgress = Math.sin(frame * 0.03) * 0.5 + 0.5;
const bodyY = lerp(0, -30, archProgress); // центр спины поднимается


Блок 2 (5–10 мин): КОШКА МОЕТСЯ
Washing — умывание: Кошка поднимает лапу → лижет лапу (голова наклоняется к лапе) → протирает ухо. Цикл: 6 сек. Повторяется много раз — очень успокаивает.
Ear wash: Лапа трёт ухо — ухо сминается → снова выпрямляется.
Tail watch: Хвост медленно извивается по-змеиному (S-кривая движется вдоль хвоста). Кошка вдруг ЗАМЕЧАЕТ хвост — смотрит на него — прыгает за ним! Крутится вокруг себя 2 раза.
Блок 3 (10–15 мин): КОШКА ХОДИТ
Walk: Кошка идёт через экран — мягкая походка. При каждом шаге — плечи мягко покачиваются. Хвост медленно колышется сзади.
Sit — stop — look: Доходит до края — садится — поворачивает голову → смотрит в другую сторону. Встаёт — идёт обратно.
Pounce position: Кошка пригибается к земле — хвост поднимается — попа покачивается. Прыгает! На что-то невидимое. Приземляется мягко — оглядывается.
Блок 4 (15–20 мин): КОШКА СПИТ
Sleep: Кошка сворачивается — голова на лапах. Дышит: бок поднимается 4 сек вверх, 4 сек вниз. ZZZ пузырьки медленно поднимаются. Ухо иногда дёргается (слышит что-то во сне).
Блок 5 (20–25 мин): КОШКА ПРЫГАЕТ
Jump sequence:
* Кошка смотрит вверх
* Пригибается (prep)
* ПРЫЖОК — вверх на 300px — arc trajectory
* Приземляется мягко — squash → stretch вверх → стабильно
* Повторяется с разных мест

ЖИВОТНОЕ 4 — РЫБКА (FISH)
ВИДЕО FISH-A — РЫБКА СОЛО (25 мин, без слов)
BPM: 60 | Mood: плавное, задумчивое
Движение рыбки:
// S-изгиб хвоста — главное движение рыбки
const fishSegments = 8; // количество сегментов тела
const waveSpeed = 0.12;
const waveAmplitude = 15;


segments.map((seg, i) => {
  const phase = i / fishSegments;
  const angle = Math.sin(frame * waveSpeed + phase * Math.PI * 2) * waveAmplitude;
  // Голова почти не двигается, хвост — максимально
  return { angle: angle * (i / fishSegments) };
});


// Движение вперёд — пропорционально интенсивности волны
const speed = waveAmplitude * waveSpeed;


Блоки:
1. Рыбка плавает в кругах по аквариуму. Пузыри из рта.
2. Рыбка подплывает к стеклу — смотрит наружу (большие глаза)
3. Два вида рыбок — плывут навстречу — разворачиваются
4. Рыбка прячется за камнем → выплывает → прячется снова
5. Ночь в аквариуме — рыбка спит (стоит почти на месте, медленно)
Аквариум-декор:
* Пузырьки всегда поднимаются снизу
* Водоросли покачиваются
* Камешки на дне
* Замок (декоративный) — рыбка проплывает через него

ЖИВОТНОЕ 6 — ПОПУГАЙ (PARROT)

Уникальная черта — КОПИРУЕТ:
Попугай копирует движения других животных/предметов. Сначала смотрит → потом повторяет (немного неточно).
Блоки:
1. Попугай качается на жёрдочке — влево-вправо — как маятник.

5. Чистит перья клювом.
Цвета попугая:
Яркие — красный, зелёный, жёлтый, синий. При хлопанье крыльями: максимальный разворот — все цвета видны.

  excited: { speed: 15, amplitude: 360, rotation: true }, // полный пропеллер
  calm:    { speed: 1.5, amplitude: 20, rotation: false }, // медленное виляние
  sad:     { speed: 0.5, amplitude: 5, rotation: false }, // поникший хвост
});


// Кошка — выгиб спины
const getCatSpinePoints = (action: string, progress: number) => {
  switch(action) {
    case 'stretch':
      return [
        { y: 0 },    // голова — не меняется
        { y: -progress * 20 }, // шея вверх
        { y: progress * 30 },  // спина вниз (выгиб)
        { y: -progress * 15 }, // зад вверх
      ];
    case 'arch':
      return [
        { y: 0 },
        { y: progress * 15 },
        { y: progress * 40 }, // максимальный выгиб вверх
        { y: progress * 15 },
      ];
  }
};


// Нос кролика — независимая анимация
const rabbitNoseWiggle = (frame: number) => ({
  x: Math.sin(frame * 0.45) * 2,
  scaleX: 1 + Math.sin(frame * 0.45) * 0.1,
});


// Рыбка — s-изгиб тела
const getFishBodyPoints = (frame: number, speed: number) => {
  return Array.from({ length: 8 }).map((_, i) => {
    const normalizedI = i / 7; // 0 (голова) до 1 (хвост)
    const waveInfluence = normalizedI * normalizedI; // голова меньше двигается
    const angle = Math.sin(frame * speed + normalizedI * Math.PI * 2) 
                  * 20 * waveInfluence;
    return { angle };
  });
};


// Хомяк — колесо
const getHamsterWheelState = (frame: number, running: boolean) => {
  const speed = running ? 8 : 0;
  const wheelAngle = (frame / 30) * speed * 360;
  const legPhase = (frame / 30) * speed * 2; // ноги в 2× быстрее
  const bodyBob = running ? Math.abs(Math.sin(legPhase)) * 4 : 0;
  return { wheelAngle, legPhase, bodyBob };
};
