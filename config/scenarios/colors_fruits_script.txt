🎬 INTRO — 0:00–1:00
Animation
* Black screen fades to bright white
* Cartoon sun rises from bottom, bouncing happily (spring animation)
* Colorful fruit characters run in from sides waving: Apple, Banana, Orange, Grape, Strawberry, Watermelon
* Title appears letter by letter: "COLORS WITH FRUITS!"
* Each letter is a different color, slight bounce on landing
* All characters jump together on the beat
Narrator (cheerful, warm female voice)
"Hello hello hello, little friends! Welcome to Colors with Fruits! Today we're going to learn SO many beautiful colors together! Are you ready? Let's go!"
Music
Upbeat, xylophone-led jingle, 120 BPM. Loopable 8-bar phrase.

🎮 BRIDGE 2 — "FIND THE COLOR!" GAME — 12:30–15:00
Animation format
* Screen shows 4 colored fruits at once
* A big colorful arrow points randomly while narrator asks
* Wrong answers: fruits shake their head + buzzer sound
* Right answer: fruit jumps, confetti, cheering
Round 1:
"Which one is RED? Look carefully… Point to the red one! … Yes! The apple is RED! Wonderful!"
Round 2:
"Which one is YELLOW? Can you find yellow? … The banana! That's right! YELLOW!"
Round 3:
"Which one is PURPLE? Hmm… Purple… Yes! The grapes are PURPLE! You're so clever!"
Round 4:
"Which one is BLUE? Blue… Blue… There it is! Blueberries are BLUE! Amazing!"
Round 5:
"Okay last one — which one is GREEN? … GREEN… Yes! The watermelon! GREEN! You did it! You're a COLOR SUPERSTAR!"
Celebration animation
* All characters run on screen cheering
* Stars, confetti, balloons fill the screen
* Text: "⭐ COLOR SUPERSTAR! ⭐"

🔄 BLOCK C — FULL REPEAT, FASTER PACE — 15:00–19:30
Same sequence as Blocks A+B but compressed — 30 seconds per color instead of 60. Faster music tempo. Characters already familiar so less intro needed.
Format per color (30 sec each):
* Color circle appears (3 sec)
* Fruit character appears (5 sec)
* Name said 3 times + character dances (15 sec)
* Text appears (5 sec)
* Quick "goodbye" to corner (2 sec)
Narrator (slightly faster, more energetic):
For each color:
"[Color]! [Fruit] is [color]! Say it with me — [COLOR]! [COLOR]! [COLOR]! Great!"
Order: Red → Yellow → Orange → Green → Pink → Purple → Blue → Brown → White

📋 REMOTION IMPLEMENTATION NOTES
Components needed
* ColorCircle — animated circle with spring bounce, color prop
* FruitCharacter — SVG cartoon character, idle/dance/wave animations
* ColorText — letter-by-letter text reveal, color prop
* IconRow — bottom row of small fruit icons that accumulate
* GameScreen — 4-fruit layout for the quiz section
* CelebrationOverlay — confetti + stars particle system
* SceneBackground — solid color bg with pulse animation
Timing system
All scenes use useCurrentFrame() and interpolate() for smooth transitions. Recommend 30fps, 1920×1080.
Audio layers
1. Background music (looped, low volume ~20%)
2. Narrator voice (main track)
3. SFX layer (pops, bounces, cheers — triggered by frame)
Font recommendation
Fredoka One or Nunito — round, friendly, highly legible for children.
