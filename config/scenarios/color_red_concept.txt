🔴 ONE CONCEPT DEEP — КРАСНЫЙ ЦВЕТ
Метод: Один концепт глубоко | Серия: Цвета Видео в серии: 3 видео × 20 минут = 60 минут про КРАСНЫЙ
________________


ПРИНЦИП МЕТОДА
Проблема всех существующих видео — они дают 10 цветов за 10 минут. Ребёнок 1–2 лет не может усвоить больше 1 концепта за раз. Нейронаука: для закрепления в долгосрочной памяти нужно 7–9 повторений с разными контекстами, с паузами между ними.
Наш подход:
* 1 видео = 1 цвет
* 20 минут = 8–12 разных контекстов одного цвета
* Диалог = ребёнок всегда даёт правильный ответ (уверенность, не тревога)
* Серия = 3 видео про один цвет прежде чем перейти к следующему

БЛОК A — ЗНАКОМСТВО С КРАСНЫМ (0:00–1:30)
Animation
* Чёрный экран
* Медленно — очень медленно — из центра расплывается красное пятно
* Растёт... растёт... заполняет весь экран
* Экран полностью красный — держится 3 секунды
* Появляется белый текст по центру: RED
* Текст пульсирует мягко
Narrator (медленно, тепло, чётко):
"Red." (пауза 2 сек) "Rееed." (пауза 2 сек) "This is red." (пауза 3 сек) "Red."
Правило этого блока:
Никаких объяснений. Никаких сравнений. Только слово и цвет. Ребёнок запоминает: ЭТОТ цвет = ЭТО слово.

БЛОК B — КРАСНЫЙ ШАР (1:30–3:30)
Animation
* Белый фон
* Из-за левого края медленно выкатывается БОЛЬШОЙ красный шар
* Катится к центру — останавливается
* Шар слегка покачивается — живой
* Текст появляется снизу: RED BALL
* Шар подпрыгивает один раз — мягко
Narrator:
"A ball." (пауза 2 сек) "A red ball." (пауза 3 сек) "The ball is red." (пауза 3 сек) "Red." (пауза 2 сек) "Red ball."
Правило этого блока:
Шар ОДИН. Фон БЕЛЫЙ. Никаких других предметов, никаких других цветов. Только красный шар на белом фоне. Максимальная концентрация.

БЛОК C — ДИАЛОГ 1: "ЭТО КРАСНЫЙ?" (3:30–5:00)
Формат диалога
Персонаж Ру — маленький круглый медвежонок — смотрит на ребёнка. Показывает красный шар. Поворачивает к камере. Спрашивает.
Animation
* Ру держит красный шар в лапах
* Смотрит на шар — потом на камеру
* Наклоняет голову — знак вопроса появляется сверху
* ПАУЗА — 4 секунды — экран почти замирает
* Шар начинает светиться — подсказка
* Ру кивает: "Да!"
Narrator:
"Is this red?" (пауза 4 секунды — время для ответа ребёнка) "Yes!" (Ру прыгает от радости) "That's right! Red! The ball is RED!" (пауза 2 сек) "Can you say red? Reeed!" (пауза 3 сек — ребёнок повторяет) "Red! Great job!"
КРИТИЧЕСКОЕ ПРАВИЛО ДИАЛОГА:
1. Вопрос ВСЕГДА имеет очевидный ответ — ребёнок не может ошибиться
2. Пауза ровно 4 секунды — не меньше (нет времени у ребёнка) и не больше (теряет нить)
3. После паузы — НЕМЕДЛЕННОЕ радостное подтверждение
4. Никогда не говорить "Неправильно" — только "Yes! Red!"
5. Дублировать ответ словом И визуально (Ру прыгает + текст RED мигает)

БЛОК D — КРАСНОЕ ЯБЛОКО (5:00–7:00)

Что меняется от шара к яблоку:
* Другой ПРЕДМЕТ — но тот же ЦВЕТ
* Ребёнок начинает понимать: красный — это про цвет, не про шар
* Это ключевой момент обобщения концепта

БЛОК E — ДИАЛОГ 2: "ПОКАЖИ КРАСНЫЙ!" (7:00–8:30)

Важно:

* Ребёнок всегда "находит правильный"
* Это НЕ тест — это подтверждение

БЛОК G — ДИАЛОГ 3: "КРАСНЫЙ ИЛИ СИНИЙ?" (10:30–12:00)
Animation
* На экране слева: красная машинка
* На экране справа: синяя машинка (та же форма — только цвет разный)
* Ру между ними — смотрит на одну, потом на другую
* Вопросительный знак
Narrator:
"Two cars!" (пауза 1 сек) "This car... is it red or blue?" (пауза 4 сек — Ру смотрит на красную машинку) "RED!" (красная машинка подпрыгивает, синяя остаётся) "That's right! This car is RED!" (пауза 2 сек) "And this one?" (Ру смотрит на синюю — но мы не изучаем синий, просто контраст) "That one is blue. But TODAY we are learning RED!" (красная машинка снова центре — большая) "RED! Red car!"
Критически важно:
* Не обучаем синему — только используем как контраст
* После вопроса про синий СРАЗУ возвращаемся к красному
* Фокус никогда не уходит от главного концепта

БЛОК J — ПЕТЛЯ-ГИМН (17:00–20:00)
Что это такое
Самая простая возможная петля — одна сцена которая зациклена. Красный фон + белый текст RED + простая мелодия. Максимально монотонно — для закрепления и засыпания.
Animation
* Красный фон пульсирует очень медленно (90% → 100% → 90% яркости)
* "RED" в центре белыми буквами — такое же пульсирование
* Иногда (каждые 20 сек) один из трёх предметов проплывает через экран
Narrator (медленно, тихо):
"Red... red... red..." (каждые 8 секунд)

ВИДЕО 2 — КРАСНЫЙ: НОВЫЕ ПРЕДМЕТЫ
Duration: 20 min | Concept: RED — расширение
Суть видео 2
Те же принципы — но новые предметы. Ребёнок уже знает красное из видео 1. Теперь укрепляем через новые контексты.
Новые предметы:
* Красная клубника (strawberry)
* Красный помидор (tomato)
* Красный пожарная машина (fire truck)
* Красный цветок (flower)
* Красная рыбка (fish)
Структура идентична видео 1 — только предметы другие
Новый элемент в видео 2: РЕАЛЬНЫЕ ФОТО
После мультяшного предмета — 5 секунд реальной фотографии. Связывает анимацию с реальным миром.
Пример:
* Анимация: красная клубника (мультяшная)
* Фото: реальная клубника на тарелке
* Narrator: "Real strawberry! Also red!"

	ИГРА 1: НАЙДИ КРАСНЫЙ (2:00–6:00)
6 раундов × 40 секунд
Каждый раунд:
1. Экран с 3 предметами (один красный, два других цветов)
2. "Can you find the red one?"
3. Пауза 4 секунды
4. Стрелка указывает на красный — конфетти
Важно: каждый раунд красный предмет на другом месте — слева, справа, в центре — чтобы ребёнок реально искал, а не просто тыкал в одно место.

ИГРА 2: КРАСНЫЙ ИЛИ НЕТ? (6:00–10:00)
8 раундов × 30 секунд
Каждый раунд:
1. Появляется один предмет
2. "Is this red?"
3. Пауза 4 секунды
4. Ответ
Чередование: 4 красных предмета и 4 нет. Когда предмет НЕ красный:
* Narrator: "No! That's blue!" (или другой цвет)
* Предмет покачивается головой
* СРАЗУ появляется красный предмет: "But THIS is red!"
* Всегда возвращаемся к красному

ИГРА 3: ЧТО ЦВЕТ? (10:00–14:00)
5 раундов × 48 секунд
Самый сложный формат — открытый вопрос.
Каждый раунд:
1. Появляется красный предмет
2. "What color is this?"
3. Пауза 5 секунд (длиннее — вопрос сложнее)
4. Ру как будто "слышит" ответ ребёнка
5. "RED! Yes! That's right! RED!"
Визуальная подсказка: Рот персонажа двигается — он как будто "говорит RED" беззвучно перед тем как сказать вслух. Ребёнок видит артикуляцию.

ФИНАЛ: РЕБЁНОК УЧИТ РУ (14:00–17:00)
Ролевая инверсия — самый эффективный момент
Animation:
* Ру притворяется что не знает что такое красный
* Смотрит на красный шар — пожимает плечами
* Смотрит в камеру — "What color is this?"
* Пауза 5 секунд
* Ру "слышит" ответ — глаза широко открываются — "RED? It's RED!"
* Прыгает — так рад что узнал!
* "You taught me! Thank you!"
Почему это работает:
Ребёнок переходит из роли "учеником" в роль "учителя". Это самый высокий уровень закрепления — объяснить другому. Создаёт ощущение компетентности и гордости.

ШАБЛОН ДИАЛОГОВЫХ ПАУЗ
ПРОСТОЙ ВОПРОС:
"Is this [color]?" → пауза 4 сек → "Yes! [COLOR]!"


ВЫБОР ИЗ ДВУХ:
"[color A] or [color B]?" → пауза 4 сек → правильный светится → "[COLOR A]!"


ОТКРЫТЫЙ ВОПРОС:
"What color is this?" → пауза 5 сек → Ру "слышит" → "[COLOR]! Yes!"


УКАЗАТЕЛЬНЫЙ:
"Show me [color]!" → пауза 4 сек → стрелка → "There it is! [COLOR]!"


ПОДТВЕРЖДЕНИЕ ПОСЛЕ ЛЮБОГО ОТВЕТА:
→ конфетти (небольшое, не взрыв)
→ Ру делает happy bounce (один прыжок, не фейерверк)
→ текст [COLOR] мигает крупно один раз
→ narrator: "Great job!"
→ снова вопрос или следующий блок

REMOTION IMPLEMENTATION
Ключевые компоненты:
// DialoguePause — самый важный компонент
interface DialoguePauseProps {
  question: string;         // текст вопроса
  pauseDuration: number;    // в кадрах (4 сек = 120 кадров)
  hintFrame: number;        // когда показывать подсказку
  answerFrame: number;      // когда давать ответ
  answer: string;           // текст ответа
  onAnswerFrame?: () => void; // конфетти + Ру прыгает
}


// ColorFlood — красный заполняет экран
const colorFlood = interpolate(
  frame,
  [0, 60],
  [0, 1],
  { easing: Easing.out(Easing.cubic) }
);
// Радиус расплывания от центра


// PauseIndicator — тихий визуальный намёк что идёт пауза
// Три точки пульсируют очень тихо в углу
const dotPulse = Math.sin(frame * 0.12 + dotIndex * 0.9) * 0.25 + 0.5;


Правило BPM для пауз:
Пауза = 4 секунды = 120 кадров при 30fps
Слишком короткая пауза (< 3 сек) = ребёнок не успевает
Слишком длинная (> 6 сек) = ребёнок теряет контекст
Идеально = 4 секунды для простых вопросов
           5 секунд для открытых вопросов
