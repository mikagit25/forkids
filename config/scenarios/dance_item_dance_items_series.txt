ГРУППА 1 — КУХНЯ (кружка, ложка, тарелка, чайник)
ВИДЕО K-A1 — КУХНЯ БЕЗ СЛОВ — СОЛО (25 мин)
BPM: 72 | Mood: тёплое, домашнее
Блок 1 (0–5 мин): КРУЖКА
Появление: Кружка медленно "приземляется" сверху — ставится на стол. Пар поднимается из неё (3–4 тонкие белые линии, drift вверх).
Движение SWAY: Кружка качается влево-вправо. Медленно — 4 сек/цикл. При наклоне влево: чай в кружке наклоняется вправо (реалистично!). При наклоне вправо: чай идёт влево.
// Жидкость в кружке — имитация
const cupTiltAngle = Math.sin(frame * swayCycleSpeed) * 12; // градусы наклона
const liquidSurface = -cupTiltAngle * 0.6; // противофаза — жидкость не успевает
// Рисуем эллипс открытия кружки с наклоном liquidSurface


Характер: спокойная, уверенная. Никогда не торопится.
________________


Блок 2 (5–10 мин): ЛОЖКА
Появление: Ложка падает сверху — с мягким звоном — стоит вертикально.
Движение МАЯТНИК: Ложка качается как маятник (держится за черенок сверху). Амплитуда: 35° в каждую сторону. В нижней точке — максимальная скорость — слегка вибрирует.
ВРАЩЕНИЕ: После 5 качаний: ложка начинает вращаться вокруг своей оси (вертикальной). Медленно — 5 сек/оборот. Выпуклая часть блестит при повороте.
Характер: изящная, звонкая. Металлический отблеск при каждом движении.
________________


Блок 3 (10–15 мин): ТАРЕЛКА
Появление: Тарелка падает плашмя (горизонтально) → приземляется → слегка bounce.
SPIN: Тарелка крутится на месте — как монетка. Сначала быстро (2 сек/оборот), потом медленнее, потом снова быстро.
ROLLING: Тарелка начинает катиться по столу — влево → вправо → обратно. При качении: видна окружность тарелки (сбоку).
Характер: круглая, стабильная, плоская.
________________


Блок 4 (15–20 мин): ЧАЙНИК
Появление: Чайник появляется с клубами пара — важно.
ПОКАЧИВАНИЕ: Медленное sway — тяжёлый (вода внутри). Амплитуда маленькая — 8°. Но! Пар всегда идёт вертикально вверх — независимо от наклона чайника.
СВИСТ: Раз в 30 сек: чайник "закипает" — трясётся 3 сек — пар валит сильнее. Потом успокаивается.
Характер: важный, тяжёлый, горячий. Самый большой из группы.
________________


Блок 5 (20–25 мин): ВСЕ ЧЕТВЕРО ВМЕСТЕ
Все 4 предмета на "кухонном столе" — в ряд.
* Чайник: большой, слева, медленный sway
* Кружка: рядом с чайником, medium sway
* Тарелка: справа, spin на месте
* Ложка: на тарелке, маятник
Волна: sway слева направо через все четыре.
________________


ВИДЕО K-A2 — КУХНЯ БЕЗ СЛОВ — ГРУППА (25 мин)
Все 4 предмета вместе с начала. Групповые движения.
Блок 1 (0–5 мин): СИНХРОННЫЙ SWAY
Все четыре качаются влево-вправо вместе. Чайник: медленно. Кружка: средне. Тарелка: spin (своё движение). Ложка: маятник. Каждый в своём ритме — но все вместе на сцене.
Блок 2 (5–10 мин): ЧАЙНИК УГОЩАЕТ ВСЕХ
Сцена: чайник наклоняется к кружке → пар (чай) "переливается". Кружка делает довольный BOB — наполнилась! Ложка прыгает в кружку — начинает помешивать (вращение в кружке). Тарелка ждёт своей очереди.
Блок 3 (10–15 мин): ВОЛНА ЧЕРЕЗ ЧЕТЫРЁХ
4 предмета в ряд — волна BOB слева направо. 3 вариации волны.
Блок 4 (15–20 мин): ХОРОВОД
4 предмета медленно вращаются по кругу — карусель. Один оборот = 12 сек. Каждый делает своё личное движение + орбита.
Блок 5 (20–25 мин): ЗАСЫПАНИЕ КУХНИ
Предметы замедляются. Чайник остывает (пар исчезает). Ложка прекращает качаться. Тарелка останавливается. Кружка делает один последний sway — и замирает. Все стоят. Тихо. Кухня спит.

ВИДЕО K-B — КУХНЯ С ОБУЧЕНИЕМ (25 мин)
Те же движения + названия + диалог
Блоки 1–4 (0–16 мин):
Каждый предмет представляется:
* Кружка: CUP / كوب / Cangkir
* Ложка: SPOON / ملعقة / Sendok
* Тарелка: PLATE / طبق / Piring
* Чайник: KETTLE / إبريق / Teko
Структура каждого (4 мин): → Появляется → название 6 раз (медленно) → действие → диалог
Диалог (пример для кружки):
"Is this a cup?" (пауза 4 сек) "YES! Cup!" "What is this?" (пауза 5 сек) "CUP!" "Where is the cup?" (2 предмета на экране, пауза 4 сек) → стрелка → "Cup!"
Блок 5 (16–20 мин): ВСЕ 4 — ДИАЛОГ
"Can you find the SPOON?" (4 предмета, пауза 5 сек) → ложка светится → "There! SPOON!" "How many cups?" (2 кружки, пауза 4 сек) → "TWO cups!"
Блок 6 (20–23 мин): ПЕСЕНКА КУХНИ
🎵 "Cup, spoon, plate and kettle! Cup, spoon, plate and kettle! On the table every day! Cup, spoon, plate and kettle!" 🎵
Блок 7 (23–25 мин): ПЕТЛЯ
Все 4 предмета медленно качаются. Тихая музыка.

ГРУППА 2 — ИГРУШКИ (мяч, кубик, пирамидка, погремушка)
ВИДЕО I-A1 — ИГРУШКИ БЕЗ СЛОВ — СОЛО (25 мин)
МЯЧ (0–6 мин):
Самое энергичное движение — bounce физика.
* 0–2 мин: прыгает вверх-вниз — 3 разных высоты
* 2–4 мин: катится влево-вправо — отскакивает от стен
* 4–6 мин: 3 мяча разного размера — разный отскок
КУБИК (6–12 мин):
Угловатый — движения с остановками.
* Tumble: кубик переворачивается по граням (90° за раз, с паузой)
* Slide: скользит плавно → резко останавливается
* Stack: 3 кубика складываются в башню → разваливаются
ПИРАМИДКА (12–18 мин):
Многоуровневая — интересный silhouette при вращении.
* Spin на месте — конус пирамиды создаёт красивый паттерн
* Wobble: наклоняется в сторону — почти падает — выпрямляется
* Кольца разлетаются — собираются обратно
ПОГРЕМУШКА (18–24 мин):
Самая шумная по характеру.
* Shake: быстрые горизонтальные движения влево-вправо
* Аудио: каждый shake → тихий звук гремения
* Арка: погремушка описывает дуги в воздухе
ВСЕ ВМЕСТЕ (24–25 мин): краткий хоровод
________________


ВИДЕО I-A2 — ИГРУШКИ БЕЗ СЛОВ — ГРУППА (25 мин)
Блок 1: МЯЧ ПРЫГАЕТ МЕЖДУ ДРУГИМИ
Кубик и пирамидка стоят. Мяч прыгает между ними — отскакивает от кубика! Кубик слегка сдвигается от удара.
Блок 2: БАШНЯ ИЗ ВСЕХ
Кубики → пирамидка → сверху мяч. Погремушка прыгает вокруг — кружится. Башня начинает покачиваться → падает → снова строится.
Блок 3: ПАРАД ИГРУШЕК
Все 4 движутся слева направо:
* Мяч: rolling + прыжки
* Кубик: tumble по граням
* Пирамидка: sliding + spin
* Погремушка: shake в воздухе
Блок 4: СИНХРОННЫЙ BOUNCE
Мяч задаёт ритм. Остальные прыгают в тот же ритм. Кубик прыгает неловко — угловато. Погремушка — легко. Пирамидка — wobble.
Блок 5: ХОРОВОД ИГРУШЕК
Все 4 вращаются по кругу. BPM 80 — бодро.

ГРУППА 3 — ОДЕВАНИЕ (ботинок, шапка, носок, варежка)
ВИДЕО O-A1 — ОДЕВАНИЕ БЕЗ СЛОВ — СОЛО (25 мин)
БОТИНОК (0–6 мин):
Самый тяжёлый. Движения основательные.
* Heavy bounce: прыгает — тяжёлый удар при приземлении
* Walk: шагает влево-вправо (поочерёдно поднимает носок и пятку)
* Stomp: несколько раз топает — вибрация экрана
* Пара ботинок: идут вместе — левый-правый-левый-правый
ШАПКА (6–12 мин):
Мягкая, лёгкая, пушистая.
* Drift: медленно парит — почти как перо
* Spin: крутится на месте — помпон болтается
* Падает на голову персонажу-предмету → надета!
* Сваливается → снова надевается
НОСОК (12–18 мин):
Самый лёгкий предмет — очень высокие прыжки.
* Высокий bounce: носок подпрыгивает до верха экрана
* Flutter: при движении немного изгибается (как ткань)
* Пара носков: один красный один синий — прыгают вместе
ВАРЕЖКА (18–24 мин):
Пушистая, мягкая — squash большой.
* Flutter: как птица машет крыльями
* Squeeze: сжимается — расправляется (как рука внутри)
* Пара варежек: "машут" пальцами
________________


ВИДЕО O-A2 — ОДЕВАНИЕ БЕЗ СЛОВ — ГРУППА (25 мин)
Блок 1: ОДЕВАНИЕ ПО ПОРЯДКУ
Последовательность как при реальном одевании:
* Носок надевается на ботинок (анимация надевания)
* Варежка "надевается" на карандаш (шутливо)
* Шапка едет вниз и накрывает кружку (из кухонной серии — камео!)
Блок 2: ПАРАД ОДЕЖДЫ
Все 4 предмета в ряд — шагают:
* Ботинок: тяжёлые шаги BOOM-BOOM
* Шапка: парит над остальными
* Носок: прыгает между ботинками
* Варежка: машет всем
Блок 3: КАРУСЕЛЬ ОДЕЖДЫ
Хоровод — все вращаются по кругу. Шапка летает сверху, остальные снизу.
Блок 4: ИГРА В ПРЯТКИ
Шапка накрывает себя одним из предметов — "спрятался"! Из-под шапки выглядывает носок — нашёлся!

ГРУППА 4 — УЧЁБА (книга, карандаш, линейка, ножницы)
ВИДЕО U-A1 — УЧЁБА БЕЗ СЛОВ — СОЛО (25 мин)
КНИГА (0–6 мин):
* Книга закрыта → медленно открывается
* Страницы перелистываются одна за другой — шелест
* Закрывается → снова открывается
* Прыгает: книга подпрыгивает и страницы разлетаются
КАРАНДАШ (6–12 мин):
* Длинный → маятник (держится за ластик сверху)
* Rolling: катится по столу слева направо
* Spin: вращается на кончике (как волчок)
* Рисует: карандаш движется и за ним остаётся цветная линия!
ЛИНЕЙКА (12–18 мин):
* Прямая и чёткая → движения угловатые, остановки чёткие
* Slide: скользит по экрану
* Вращение: 180° и обратно
* Измеряет: прикладывается к другим предметам (сравнение размеров)
НОЖНИЦЫ (18–24 мин):
* Открываются-закрываются: SNIP SNIP SNIP
* Tanzen: ножницы "танцуют" открываясь и закрываясь в ритм
* Режут лист бумаги: бумага распадается на два листа
________________


ВИДЕО U-A2 — УЧЁБА БЕЗ СЛОВ — ГРУППА (25 мин)
Блок 1: КАРАНДАШ РИСУЕТ
Карандаш движется → рисует книгу. Книга появляется! Карандаш рисует линейку. Линейка появляется. Карандаш рисует ножницы. Ножницы появляются.
Блок 2: ЛИНЕЙКА ИЗМЕРЯЕТ ВСЕХ
Линейка прикладывается к каждому: Карандаш — длинный. Ножницы — длиннее. Книга — широкая. Линейка — самая прямая.
Блок 3: НОЖНИЦЫ РЕЖУТ БУМАГУ
Длинная полоса бумаги — ножницы режут её на кусочки. Кусочки разлетаются. Карандаш рисует на кусочках звёздочки.
Блок 4: ВСЕ ВМЕСТЕ — УРОК
Книга открыта. Карандаш "пишет". Линейка "ведёт" прямые линии. Ножницы ждут. Гармоничная рабочая сцена.

ГРУППА 5 — ВАННАЯ (зубная щётка, мыло, полотенце, расчёска)

ЗУБНАЯ ЩЁТКА (0–6 мин):
* Быстрое движение вверх-вниз (чистит зубы в воздухе)
* Shake: вибрирует — как электрическая
* Прыгает бодро: маленькая и лёгкая
МЫЛО (6–12 мин):
* СКОЛЬЗКОЕ! Не стоит спокойно.
* Slide: постоянно куда-то уезжает (easing out — скользит)
* Escape: убегает от рук (от краёв экрана)
* Bounce: падает — скользит по полу
ПОЛОТЕНЦЕ (12–18 мин):
* Гибкое! Wave по всей длине
* Flutter: как флаг на ветру
* Fold: складывается пополам (сгибается)
РАСЧЁСКА (18–24 мин):
* Движения вверх-вниз (расчёсывает воздух)
* Spin: вращается — зубья создают визуальный паттерн
* Маятник: держится за ручку — качается

ГРУППА 6 — ДОМ (ключ, зонт, лампа, телефон)
ВИДЕО D-A1 — ДОМ БЕЗ СЛОВ — СОЛО (25 мин)
КЛЮЧ (0–6 мин):
* Качается на верёвочке — маятник
* При каждом качании: тихий металлический звон
* Вращается — замысловатая форма видна при вращении
* Несколько ключей: связка ключей — все покачиваются вместе
ЗОНТ (6–12 мин):
* ГЛАВНЫЙ МОМЕНТ: раскрывается! (scale 0.3 → 1.0, радиально)
* Закрывается. Снова раскрывается.
* Раскрытый: spinning — зонт крутится как пропеллер
* Закрытый: маятник
ЛАМПА (12–18 мин):
* Pulse: светится — пульсирует яркость
* При включении: вспышка → ровный свет
* Покачивается — свет качается вместе с лампой
* "Мигает" (SOS ритм — игриво)
ТЕЛЕФОН (18–24 мин):
* Vibrate: вибрирует (высокочастотный shake)
* Bounce: прыгает при "звонке"
* Spin: вращается (как волчок)

ВИДЕО D-B — ДОМ С ОБУЧЕНИЕМ (25 мин)

ГРУППА 7 — МУЗЫКА (барабан, колокольчик, дудочка, ксилофон)
ВИДЕО M-A1 — МУЗЫКА БЕЗ СЛОВ — СОЛО (25 мин)
БАРАБАН (0–6 мин):
* BOUNCE НА БИТ: каждый такт — удар по барабану (bounce мembrane)
* Барабан дрожит после каждого удара (ripple на мембране)
* BPM 80 — бодрый ритм
* Разные части: удар по центру / по краю (разный звук)
КОЛОКОЛЬЧИК (6–12 мин):
* Swing: качается → в нижней точке — ЗВОН
* Очень лёгкий — drift при малейшем движении
* Несколько колокольчиков: разные размеры → разные ноты
ДУДОЧКА (12–18 мин):
* Маятник: держится за конец — качается
* "Играет": вверх-вниз движения — воображаемые пальцы
* Музыкальные ноты вылетают из раструба
КСИЛОФОН (18–24 мин):
* Полоски разного цвета светятся по очереди
* Небольшой молоточек ударяет по каждой полоске
* Цвет → нота → волна расходится от удара
* Гамма: до ре ми фа соль ля си до — слева направо

ВИДЕО M-A2 — МУЗЫКА БЕЗ СЛОВ — ОРКЕСТР (25 мин)
Все 4 инструмента вместе — музыкальный ансамбль
Блок 1: НАСТРОЙКА
* Каждый инструмент "пробует" свой звук по очереди
* Барабан: BOOM BOOM. Колокольчик: дзинь. Дудочка: тю-тю. Ксилофон: до-ре-ми
Блок 2: ВМЕСТЕ — ПРОСТАЯ МЕЛОДИЯ
* Ксилофон играет "Twinkle Twinkle" (полоски светятся)
* Барабан: отбивает такт
* Колокольчик: акценты
* Дудочка: держит ноту
Блок 3: ДИРИЖЁР-ЛОЖКА (камео из кухонной серии!)
* Ложка появляется → начинает дирижировать
* Инструменты следуют за ней
Блок 4: СОЛО КАЖДОГО
* По 30 сек соло у каждого — остальные тихо качаются
Блок 5: ФИНАЛ-ХОРОВОД
* Все 4 в хороводе + музыкальные ноты летают вокруг

4 ФИНАЛЬНЫХ МИКС-ВИДЕО
ФИНАЛ 1 — ВСЯ КУХНЯ + ВСЕ ИГРУШКИ (8 предметов, 25 мин)
Две группы на одном экране. Кухня слева, игрушки справа. Мяч прыгает и попадает в кружку → кружка смеётся. Кубики складываются рядом с тарелками — сравнение размеров.
ФИНАЛ 2 — ОДЕВАНИЕ + ВАННАЯ (8 предметов, 25 мин)
Утренняя рутина: ботинки надеваются → зубная щётка чистит → мыло пенится → полотенце вытирает → шапка надевается → все уходят. Нарративная последовательность без слов.
ФИНАЛ 3 — УЧЁБА + МУЗЫКА (8 предметов, 25 мин)
Карандаш рисует ноты → ксилофон играет по ним. Ножницы режут бумагу → бумага превращается в барабан. Линейка "дирижирует" всем оркестром.
ФИНАЛ 4 — БОЛЬШОЙ ПАРАД ВСЕХ (21 предмет, 30 мин)
Все предметы из всех групп входят на экран по одному. Встают в большой хоровод. Каждый делает своё характерное движение. Самое праздничное видео всей серии.

const getLiquidSurface = (cupAngle: number) => {
  const lag = cupAngle * -0.6; // жидкость отстаёт от наклона
  return lag; // угол поверхности жидкости
};


// Umbrella open/close
const getUmbrellaRadius = (frame: number, config: UmbrellaConfig) => {
  const phase = (frame % config.cycleDuration) / config.cycleDuration;
  if (phase < 0.3) return interpolate(phase, [0, 0.3], [minR, maxR], { easing: Easing.out(Easing.back(1.5)) });
  if (phase < 0.7) return maxR; // открыт
  return interpolate(phase, [0.7, 1.0], [maxR, minR]);
};
