🕺 ТАНЦУЮЩИЕ ФИГУРЫ — Серия без слов
Формат: Remotion | Возраст: 0–3 года | Слова: НЕТ Цель: удержание внимания, watch time, фоновый просмотр Длина: 20–30 мин каждое | Серия: 12 видео
________________


ПРИНЦИПЫ СЕРИИ
Без слов — только музыка и движение Без резких движений — всё плавное, предсказуемое Без вспышек — переходы цвета всегда медленные (минимум 2 сек) Повторяющийся паттерн — ребёнок предугадывает следующее движение → спокойствие Satisfying завершённость — каждый цикл движения заканчивается "правильно"
________________

ВИДЕО 1 — ОДИН КРУГ (самое простое)
Duration: 25 min | Shape: Circle only | Count: 1
Структура (5 блоков × 5 мин):
Блок 1 (0–5 мин): BOB
* Один большой красный круг в центре
* Медленно поднимается и опускается — 3 сек вверх, 3 сек вниз
* Фон тёмно-синий
* BPM 70 — каждый подъём на такт
Блок 2 (5–10 мин): SWAY
* Тот же круг — теперь качается влево-вправо
* Как маятник — плавный easing
* Цвет плавно переходит: красный → оранжевый
Блок 3 (10–15 мин): SPIN
* Круг медленно вращается на месте
* Одна сторона чуть светлее — вращение видно
* Цвет: оранжевый → жёлтый
Блок 4 (15–20 мин): PULSE
* Круг дышит — расширяется и сжимается
* Как дыхание: 4 сек расширение, 4 сек сужение
* Цвет: жёлтый → зелёный
Блок 5 (20–25 мин): DRIFT
* Круг медленно плывёт по экрану
* Меняет направление плавно — никогда резко
* Цвет: зелёный → синий → фиолетовый
Remotion код:
// BOB
const bobY = Math.sin(frame * (2 * Math.PI) / (bobPeriod * 30)) * bobAmplitude;


// SWAY
const swayX = Math.sin(frame * (2 * Math.PI) / (swayPeriod * 30)) * swayAmplitude;


// SPIN (для круга не видно — лучше добавить полосу на 1/4 диаметра)
const spinAngle = (frame / (spinPeriod * 30)) * 360;


// PULSE
const pulse = 1 + Math.sin(frame * (2 * Math.PI) / (pulsePeriod * 30)) * 0.15;


// DRIFT (Lissajous для красивого пути)
const driftX = cx + Math.sin(frame * 0.008) * (screenW * 0.35);
const driftY = cy + Math.sin(frame * 0.005 + 1) * (screenH * 0.3);

ВИДЕО 2 — ТРИ КРУГА
Duration: 25 min | Shape: Circle | Count: 3
Блок 1: СИНХРОННЫЙ BOB
* 3 круга в ряд — красный, синий, жёлтый
* Все прыгают ОДНОВРЕМЕННО вверх-вниз
* Очень чёткий ритм
Блок 2: ВОЛНА (WAVE)
* Те же 3 круга
* Левый прыгает → через 0.3 сек средний → через 0.3 сек правый
* Волна идёт слева направо
* Потом справа налево
* Потом снова слева направо — бесконечно
const waveDelay = 0.3 * 30; // 9 кадров
const ball1Y = Math.sin(frame * speed) * amp;
const ball2Y = Math.sin((frame - waveDelay) * speed) * amp;
const ball3Y = Math.sin((frame - waveDelay * 2) * speed) * amp;


Блок 3: РАЗНЫЕ РИТМЫ
* Левый прыгает быстро (1.5 сек/цикл)
* Средний медленно (3 сек/цикл)
* Правый очень медленно (5 сек/цикл)
* Паттерны случайно совпадают → редкий момент синхрона → дети замечают!
Блок 4: ОРБИТА
* Средний круг в центре — стоит
* Два других медленно облетают его по кругу
* Один по часовой, другой против
* Иногда почти сталкиваются — дети замирают
Блок 5: DRIFT ВМЕСТЕ
* Все три дрейфуют по экрану
* Сохраняют расстояние между собой
* Плавные неторопливые пути

ВИДЕО 3 — ПЯТЬ КРУГОВ (волновая серия)
Duration: 25 min | Shape: Circle | Count: 5
Это самое гипнотическое видео серии — маятниковая волна
Блок 1: ИДЕАЛЬНАЯ ВОЛНА Пять кругов с задержкой 0.4 сек — классическая волна. Цвета радуги: красный, оранжевый, жёлтый, зелёный, синий.
Блок 2: ОБРАТНАЯ ВОЛНА Та же волна справа налево.
Блок 3: КОНКУРИРУЮЩИЕ ВОЛНЫ Волна идёт одновременно с обеих сторон — встречаются в центре. Moment of collision: все 5 синхронизируются на 1 сек — потом расходятся.
Блок 4: ВЕРТИКАЛЬНАЯ ВОЛНА Те же 5 кругов — но теперь в столбик, волна сверху вниз.
Блок 5: ПУЛЬС + ВОЛНА Каждый круг пульсирует (PULSE) + волна по цвету (не по движению). Цвет переходит слева направо как бегущая строка.
________________


ВИДЕО 4 — КВАДРАТЫ МАРШ
Duration: 25 min | Shape: Square | Count: 4
Квадраты отличаются от кругов — они ПОВОРАЧИВАЮТСЯ. Это видно. Вращение квадрата = совсем другое визуальное ощущение.
Блок 1: МАРШ
* 4 квадрата в ряд идут влево → достигают края → телепортируются вправо → снова идут
* Как бесконечный марш
* Каждый шаг — маленький подпрыжок (Bob на каждом шаге)
* BPM 90 — бодрый марш
Блок 2: ВРАЩЕНИЕ
* 4 квадрата стоят и медленно вращаются
* Каждый в свою сторону: 1 и 3 по часовой, 2 и 4 против
* Скорость вращения разная — создаёт красивый хаос-паттерн
Блок 3: СИНХРОННЫЙ ПОВОРОТ
* Все 4 квадрата вращаются ОДНОВРЕМЕННО в одну сторону
* Очень организованно и красиво
Блок 4: ДИАГОНАЛЬ
* Квадраты расставлены не в ряд а по диагонали (лесенка)
* Волна идёт по диагонали
Блок 5: ORBIT + SPIN
* 1 квадрат в центре медленно вращается
* 3 квадрата облетают его по кругу + сами вращаются
* Как планетная система квадратов
________________


ВИДЕО 5 — ТРЕУГОЛЬНИКИ
Duration: 25 min | Shape: Triangle | Count: 3
Треугольники интереснее всего при вращении — форма меняется визуально при повороте.
Блок 1: КАЧАНИЕ
* 3 треугольника в ряд
* Качаются влево-вправо как маятники
* При наклоне влево — острый угол смотрит влево, и наоборот
* Очень organic движение
Блок 2: ПЕРЕВОРОТ
* Треугольник медленно поворачивается на 180° — теперь острым вниз
* Снова на 180° — острым вверх
* Цикл: 4 сек вращение → 3 сек стоп в новой позиции → снова
Блок 3: ПИРАМИДА → РАЗЛЁТ
* 3 треугольника складываются в пирамиду
* Пауза 2 сек
* Разлетаются в стороны
* Плавают отдельно
* Снова слетаются в пирамиду
* Цикл
Блок 4: SPINNING TRIO
* 3 треугольника вращаются по большому кругу вокруг центра
* Каждый дополнительно вращается вокруг себя
* Паттерн как у пропеллера
Блок 5: ВОЛНА ТРЕУГОЛЬНИКОВ
* Треугольники наклоняются волной
* При наклоне форма кажется другой — красивый эффект
________________


ВИДЕО 6 — МИКС ФИГУР (круг + квадрат + треугольник)
Duration: 25 min | Shape: Mixed | Count: 3 (по одной каждой)
Блок 1: ЗНАКОМСТВО
* Каждая фигура появляется по очереди — входит с разных сторон
* Круг катится, квадрат шагает, треугольник скользит
* Встречаются в центре — стоят рядом
Блок 2: СИНХРОННЫЙ BOB
* Все три прыгают вместе
* Но РАЗНЫЕ фигуры — красивый контраст форм в движении
Блок 3: ОБМЕН МЕСТАМИ
* Круг и квадрат меняются местами (дуга сверху)
* Треугольник остаётся в центре
* Теперь круг и треугольник меняются
* И так по кругу — танец обмена
Блок 4: КАЖДЫЙ ПО-СВОЕМУ
* Круг: DRIFT по экрану (плавает)
* Квадрат: MARCH (идёт прямо)
* Треугольник: SPIN на месте
* Все три делают своё — красивый хаос
Блок 5: ФИНАЛЬНЫЙ ТАНЕЦ
* Все три движутся по большому кругу вокруг центра
* Меняются цветами плавно
* Гармоничное завершение
________________


ВИДЕО 7 — МНОГО ФИГУР (фоновый/медитативный)
Duration: 30 min | Shape: Mixed | Count: 7–12 на экране
Концепция
Экран почти заполнен фигурами — но не тесно. Каждая движется по своему пути — никогда не сталкиваются. Как аквариум — бесконечно смотришь.
Правила движения:
* Каждая фигура имеет свой Lissajous путь
* Скорость медленная — самая быстрая 3 сек/цикл
* Никаких столкновений (bounding box проверка)
* Цвета меняются независимо и очень медленно (20 сек на цвет)
Фигуры:
* 4 круга (разный размер)
* 3 квадрата (разный размер, вращаются медленно)
* 3 треугольника (качаются)
* 2 звезды (пульсируют)
Блок структура (30 мин):
* Мин 0–10: фигуры появляются по одной (по 50 сек каждая) — экран заполняется постепенно
* Мин 10–20: все на экране — максимальный аквариум
* Мин 20–30: фигуры по одной уходят за край — экран пустеет
Это идеальный фоновый контент — родители включают пока готовят.
________________


ВИДЕО 8 — ЗВЁЗДЫ
Duration: 25 min | Shape: Star | Count: 1–5
Блок 1: ОДНА ЗВЕЗДА МЕРЦАЕТ
* Большая жёлтая звезда пульсирует (PULSE)
* При расширении — чуть светлее (но без glow)
* BPM 60 — очень медленно
Блок 2: ЗВЕЗДА ВРАЩАЕТСЯ
* Медленное вращение — 8 сек/оборот
* Лучи звезды создают красивый паттерн при повороте
* Цвет меняется: жёлтый → оранжевый → красный
Блок 3: ПАДАЮЩИЕ ЗВЁЗДЫ
* 3 звезды медленно падают сверху вниз
* При достижении дна — плавно появляются сверху снова
* Разная скорость: медленная, средняя, очень медленная
* Как снег — только звёзды
Блок 4: ОРБИТА ЗВЁЗД
* 4 маленькие звезды облетают 1 большую в центре
* Разные орбиты и скорости
* Как солнечная система
Блок 5: НОЧНОЕ НЕБО
* 5 звёзд разного размера дрейфуют по тёмному экрану
* Очень медленно — почти неподвижны
* Мерцание независимое у каждой
* Идеально для засыпания

ВИДЕО 9 — СЕРДЦА (эмоциональное)
Duration: 25 min | Shape: Heart | Count: 1–3
Блок 1: ОДНО СЕРДЦЕ ДЫШИТ
* Большое красное сердце — PULSE синхронизировано с BPM 60
* Как настоящее сердцебиение — но очень мягко
* Тёплый фон
Блок 2: ДВА СЕРДЦА ВСТРЕЧАЮТСЯ
* Два сердца появляются с разных сторон
* Медленно движутся к центру
* Почти встречаются — касаются — отплывают
* Возвращаются — снова к центру — цикл
Блок 3: СЕРДЦА ЛЕТЯТ
* 3 сердца медленно поднимаются снизу вверх
* Как мыльные пузыри — с покачиванием
* Достигают верха — растворяются — снизу новые
Блок 4: КАЧАНИЕ
* 3 сердца в ряд качаются (SWAY)
* Каждое своей амплитудой и частотой
* Создаётся organic живой паттерн
Блок 5: КРУЖЕНИЕ
* 2 сердца кружатся вместе — как танцоры
* Расходятся → сближаются → снова расходятся
* Очень лирично и нежно

ВИДЕО 10 — ЧЕРЕДОВАНИЕ ЦВЕТОВ
Duration: 30 min | Shape: Circle | Count: 6
Главная идея
6 кругов в два ряда (3×2). Каждый круг меняет цвет независимо. Паттерны смены цветов создают визуальные "узоры во времени".
Блок 1: СЛУЧАЙНАЯ СМЕНА
* Каждый круг меняет цвет в своё время (независимо)
* Смена происходит каждые 3–7 сек (разная для каждого)
* Всегда плавный переход 2 сек
* Эффект: живой пульсирующий коврик
Блок 2: ВОЛНА ЦВЕТА
* Цвет меняется волной слева направо
* Все кружки становятся красными → слева начинается синий → волна синего идёт вправо
* Потом желтая волна, зелёная волна
Блок 3: ШАХМАТНЫЙ ПАТТЕРН
* Чётные кружки красные, нечётные синие
* Потом меняются: красные становятся синими и наоборот
* Медленная пульсация шахматного паттерна
Блок 4: РАДУГА СЛЕВА НАПРАВО
* Кружки выстраиваются в радугу: красный-оранжевый-жёлтый-зелёный-синий-фиолетовый
* Потом смещаются: второй становится красным, первый — фиолетовым
* Радуга "едет" влево
Блок 5: ВСЕ ВМЕСТЕ
* Кружки движутся (BOB) + меняют цвет
* Самое насыщенное и красивое — финал видео
________________


ВИДЕО 11 — БОЛЬШОЙ И МАЛЕНЬКИЙ
Duration: 25 min | Shape: Mixed | Count: 2 (пара)
Каждый блок — пара фигур: одна большая, одна маленькая
Блок 1: Большой круг и маленький круг. Маленький прыгает быстрее — маленьким легче! Блок 2: Большой квадрат вращается медленно. Маленький вращается быстро — и наматывает круги вокруг большого. Блок 3: Маленький треугольник сидит на большом и вместе качаются (SWAY). Блок 4: Большой DRIFT по экрану. Маленький следует за ним на расстоянии — как тень. Блок 5: Оба PULSE но в противофазе — когда один расширяется другой сжимается.
________________


ВИДЕО 12 — НОЧНОЙ (для засыпания)
Duration: 30 min | Shape: Mixed | Count: 3–5
Особые правила:
* Тёмный фон (тёмно-синий #050A1A)
* Все фигуры мягкие светлые цвета (не насыщенные): пастельные
* Движение МАКСИМАЛЬНО медленное — в 2 раза медленнее обычного
* BPM 50–55
* Яркость постепенно уменьшается к концу (100% → 60% за 30 мин)
Блок 1 (0–8 мин): BREATHING
* 3 пастельных круга — все вместе дышат (PULSE)
* Цикл 10 сек — точно как дыхание засыпающего ребёнка
* Синхронизирует дыхание
Блок 2 (8–16 мин): DRIFT
* Фигуры медленно дрейфуют — почти неподвижны
* Каждую минуту одна фигура чуть меняет направление
* Как медузы в тёмной воде
Блок 3 (16–22 мин): FADE IN-OUT
* Фигуры медленно появляются и исчезают
* Появление 5 сек — держится 10 сек — исчезновение 5 сек
* По 1-2 на экране в любой момент
Блок 4 (22–28 мин): ПОЧТИ СТОП
* Очень медленное вращение одного круга
* Остальные исчезают
* BPM 50
Блок 5 (28–30 мин): ФИНАЛЬНОЕ ЗАТУХАНИЕ
* Последний круг очень медленно уменьшается
* За 2 минуты исчезает совсем
* Экран темнеет

// Глобальные константы серии
const SHAPES_CONFIG = {
  background: {
    day:    '#0A1628', // тёмно-синий
    night:  '#050A1A', // очень тёмный для ночного видео
  },
  motion: {
    minPeriod: 1.5,   // сек — быстрейшее движение
    maxPeriod: 12.0,  // сек — медленнейшее
    nightMultiplier: 2.0, // всё в 2 раза медленнее для ночного
  },
  colorTransition: {
    minDuration: 2.0,  // сек — минимальная смена цвета
    normalDuration: 4.0,
    nightDuration: 8.0,
  },
  easing: Easing.inOut(Easing.sine), // для ВСЕХ движений — никогда Linear
};

// ShapeRenderer — рендер любой фигуры
const ShapeRenderer = ({ shape, x, y, size, color, rotation }) => {
  switch(shape) {
    case 'circle':
      return <circle cx={x} cy={y} r={size/2} fill={color}/>;
    case 'square':
      return <rect x={x-size/2} y={y-size/2} width={size} height={size}
               fill={color}
               transform={`rotate(${rotation} ${x} ${y})`}/>;
    case 'triangle':
      const h = size * 0.866;
      return <polygon
               points={`${x},${y-h*0.67} ${x-size/2},${y+h*0.33} ${x+size/2},${y+h*0.33}`}
               fill={color}
               transform={`rotate(${rotation} ${x} ${y})`}/>;
    case 'star':
      return <StarPath cx={x} cy={y} outerR={size/2} innerR={size/4}
               points={5} fill={color}
               transform={`rotate(${rotation} ${x} ${y})`}/>;
    case 'heart':
      return <HeartPath cx={x} cy={y} size={size} fill={color}
               transform={`rotate(${rotation} ${x} ${y})`}/>;
  }
};


// MotionCalculator — вычисляет позицию для каждого типа движения
const getMotion = (type, frame, config) => {
  const t = frame / 30; // в секундах
  switch(type) {
    case 'BOB':
      return { dy: Math.sin(t * 2*Math.PI / config.period) * config.amplitude };
    case 'SWAY':
      return { dx: Math.sin(t * 2*Math.PI / config.period) * config.amplitude };
    case 'SPIN':
      return { rotation: (t / config.period) * 360 };
    case 'DRIFT':
      return {
        dx: Math.sin(t * config.freqX) * config.radiusX,
        dy: Math.sin(t * config.freqY + config.phase) * config.radiusY,
      };
    case 'PULSE':
      return { scale: 1 + Math.sin(t * 2*Math.PI / config.period) * config.amount };
    case 'WAVE':
      return { dy: Math.sin((t - config.shapeIndex * config.delay) * 2*Math.PI / config.period) * config.amplitude };
    case 'ORBIT':
      const angle = (t / config.period) * 2 * Math.PI + config.phase;
      return {
        dx: Math.cos(angle) * config.radius,
        dy: Math.sin(angle) * config.radius,
      };
  }
};


// ColorTransition — плавная смена цвета
const useColorTransition = (colors, cycleSeconds, frame) => {
  const totalFrames = cycleSeconds * 30;
  const cycleFrame = frame % (totalFrames * colors.length);
  const colorIndex = Math.floor(cycleFrame / totalFrames);
  const progress = (cycleFrame % totalFrames) / totalFrames;
  const easedProgress = Easing.inOut(Easing.sine)(progress);
  return lerpColor(colors[colorIndex], colors[(colorIndex + 1) % colors.length], easedProgress);
};
