ВИДЕО 5.1 — ДЕНЬ И НОЧЬ
Duration: 30 min | Cycle: 10 min × 3 | BPM: 58
Концепция
Одно дерево с фруктами. Свет меняется от утра до ночи и обратно. Фрукты "просыпаются" днём и "засыпают" ночью. Идеальное видео для укладывания спать.
Цикл 10 минут:
Фаза 1 — РАССВЕТ (0:00–2:00)
Фон: медленно светлеет — тёмно-синий → серо-фиолетовый → розово-оранжевый
* Небо: interpolate 8 цветов за 2 мин
* Горизонт: первые золотистые лучи
* Дерево: силуэт тёмный → постепенно видны детали
Фрукты:
* Начинают с "закрытого" состояния (немного сложены, тусклые)
* По одному "открываются" как цветы (scale 0.8 → 1.0)
* Первый листик: начинает тихое движение (BOB 5 сек/цикл)
Фаза 2 — УТРО (2:00–4:00)
Фон: голубое небо, солнце появляется (жёлтый круг, медленно поднимается)
* Птица пролетает (простой силуэт, слева направо)
* Тени: длинные, уходят влево
Фрукты:
* Все активны — медленный BOB (3 сек/цикл)
* Яркие цвета — максимальная насыщенность
* Листья: BOB в разных фазах (живое дерево)
Фаза 3 — ПОЛДЕНЬ (4:00–6:00)
Фон: ярко-голубое небо, солнце в зените
* Тени: короткие, прямо под фруктами
* Самые яркие цвета
Фрукты:
* Чуть медленнее — жаркий полдень
* Одно яблоко: PULSE (дышит от жары)
Фаза 4 — ВЕЧЕР (6:00–8:00)
Фон: золотой → оранжевый → розовый
* Солнце опускается, длинные тени вправо
Фрукты:
* Движения замедляются: BOB 6 сек → 8 сек/цикл
* Насыщенность падает: 100% → 80%
* "Усталые" — не такие бодрые
Фаза 5 — ЗАКАТ (8:00–9:00)
Фон: тёмно-оранжевый → фиолетовый → тёмно-синий
* Солнце скрывается. Звезда появляется (одна)
Фрукты:
* Медленно "закрываются" (scale 1.0 → 0.85)
* Цвета тускнеют: насыщенность 80% → 50%
* Движения еле-еле
Фаза 6 — НОЧЬ (9:00–10:00)
Фон: тёмно-синий (#0A0A1F), луна (белый полукруг), звёзды
* Тихо, никакого движения фона
Фрукты:
* Почти стоят — движение 0.5px
* Тусклые, почти силуэты
* Одно яблоко: медленный PULSE как дыхание (10 сек/цикл)
* ZZZ — крошечные пузырьки из одного фрукта (очень тихо)
// Система дня-ночи
const DAY_CYCLE_FRAMES = 10 * 60 * 30; // 10 минут


const getSkyColor = (dayProgress: number) => {
  const COLORS = [
    { p: 0.00, color: [10,  10,  31] },  // глубокая ночь

    { p: 0.20, color: [255, 140, 100] }, // рассвет

    { p: 0.50, color: [100, 180, 240] }, // полдень

ВИДЕО 5.2 — ЧЕТЫРЕ СЕЗОНА — ОДНО ДЕРЕВО
Duration: 30 min | Cycle: 7.5 min × 4 | BPM: 60
Концепция
Одно дерево стоит в центре. Сезоны меняются вокруг него. Каждый сезон — 7.5 минут. Три полных цикла за 30 минут.
Дерево-персонаж:
* Ствол: тёмно-коричневый, всегда виден
* Ветки: вверх и в стороны — стиль без листьев виден зимой
* На ветках: фрукты (появляются летом, опадают осенью)
________________


ВЕСНА (0:00–7:30)
Фон: светло-голубой, пастельный, мягкий
* Земля: серо-коричневая → начинает зеленеть
Трансформация дерева:
* Минута 1–2: на кончиках веток появляются крошечные бутоны (точки)
* Минута 2–4: бутоны раскрываются в розово-белые цветы (scale 0 → 1)
* Минута 4–6: цветы опадают лепестками (DRIFT вниз)
* Минута 6–7.5: появляются первые крошечные зелёные листики
Эффекты:
* Лепестки: 20–30 штук, медленно падают (0.5 px/frame)
* Птица прилетает и садится на ветку
* Бабочка появляется на цветке
________________


ЛЕТО (7:30–15:00)
Фон: яркий голубой, насыщенный
Трансформация дерева:
* Листья полностью выросли — густая зелёная крона
* На ветках: появляются маленькие зелёные фрукты
* Зреют: зелёный → жёлтый → красный (постепенно за 4 мин)
* К концу: 8–10 спелых красных яблок на ветках
Движение:
* Листья колышутся (WAVE через крону)
* Яблоки BOB медленно
* Пчела вьётся вокруг одного яблока
________________


ОСЕНЬ (15:00–22:30)
Фон: оранжево-золотой → тёмно-красный
Трансформация дерева:
* Листья меняют цвет: зелёный → жёлтый → оранжевый → красный → коричневый
* Каждый лист меняется немного по-своему
* Листья начинают опадать — спиральное падение
Яблоки:
* Одно за другим отрываются и падают (bounce на земле)
* К концу: все яблоки упали, лежат под деревом
Падение листьев:
const leaf = {
  x: startX,
  y: startY,
  rotation: 0,
  vx: random(-1, 1),
  vy: 0.3,
  rotationSpeed: random(-3, 3),
};
// При падении: медленное вращение + горизонтальное покачивание
leaf.x += leaf.vx + Math.sin(frame * 0.05) * 0.5;
leaf.y += leaf.vy;
leaf.rotation += leaf.rotationSpeed;


________________


ЗИМА (22:30–30:00)
Фон: бело-серый, холодный, синеватый
Трансформация дерева:
* Все листья опали — голые ветки
* Медленно начинает падать снег
Снег:
const snowflake = {
  x: random(0, screenW),
  y: -10,
  size: random(3, 8),
  fallSpeed: random(0.3, 0.8),
  wobble: random(0.02, 0.05),
};
snowflake.x += Math.sin(frame * snowflake.wobble) * 1.5;
snowflake.y += snowflake.fallSpeed;


* Снег накапливается на ветках (белый слой растёт)
* Земля покрывается белым
* Всё белее и тише
* К концу зимы: намёк на рассвет → переход к весне

ВИДЕО 5.3 — ДОЖДЬ В САДУ
Duration: 30 min | Cycle: 6 min × 5 | BPM: 55
Концепция
Сад с фруктами и овощами. Начинается дождь. Растения тянутся к каплям, пьют воду, немного подрастают. Потом дождь стихает. Сад блестит от воды.
Структура сада:
* Фон: земля с грядками
* 5–6 растений разного размера: морковь, яблоня, помидор, клубника, подсолнух
* Каждое растение в своей "грядке"
Детальный цикл (6 мин):
Фаза 1 — СУХО (0:00–1:00)
* Сад без дождя
* Растения чуть поникшие (thirsty: scale 0.95, наклон)
* Небо: серовато-белое, облака
Фаза 2 — ПЕРВЫЕ КАПЛИ (1:00–1:30)
* Одна капля падает на лист — расходятся круги
* Потом ещё одна
* Растения "замечают" — чуть приподнимаются
Фаза 3 — ДОЖДЬ (1:30–3:30)
* Полноценный дождь (50–70 капель на экране)
* Капли падают на листья и плоды:
   * На плоском листе: круги расходятся
   * На яблоке: капля скатывается вниз
   * В луже под растением: плюх + кольца
Растения под дождём:
* Морковь: листва оживает, встаёт прямее
* Яблоня: листья принимают капли, блестят
* Клубника: клонится под весом капель, потом выпрямляется
Рост:
* Очень медленный прирост высоты: +0.5px/сек на каждое растение
* За 2 мин дождя: +60px — заметно!
Фаза 4 — ДОЖДЬ СТИХАЕТ (3:30–4:30)
* Капли становятся реже: 70 → 30 → 10 → 3 → 0
* Последняя капля: медленно, торжественно
Фаза 5 — ПОСЛЕ ДОЖДЯ (4:30–6:00)
* Всё блестит (shine highlights на каждом листе)
* Радуга появляется в небе (маленькая, краткая — 30 сек)
* Растения полностью выпрямились — максимально живые
* Капли медленно стекают с листьев
// Капля дождя на листе
const leafRipple = {
  x: impactX,
  y: impactY,
  radius: 0,
  opacity: 0.6,
};
// Кольца расширяются
leafRipple.radius += 1.5;
leafRipple.opacity *= 0.96;


// Капля скатывается с яблока
const dropOnApple = {
  angle: 0, // начало сверху яблока
  radius: appleRadius,
};
dropOnApple.angle += 0.02; // скатывается по поверхности


Звуковые слои:
* Тяжёлые капли: глубокий plonk
* Лёгкие капли: высокий tink
* Капли в лужу: бульк
* Капли на листья: шелест
* После дождя: птица запела (первая)

ВИДЕО 5.4 — ВЕТЕР КАЧАЕТ
Duration: 25 min | Cycle: 5 min × 5 | BPM: 58
Концепция
Фрукты на ветках. Ветер дует — то слабый, то сильный. Каждый порыв ветра — новая сила и направление. Иногда один фрукт отрывается и падает.
Система ветра:
// Ветер — медленно изменяющийся сигнал
let windStrength = 0; // 0-1
let windDirection = 0; // градусы


// Случайные порывы
const targetWind = random(0, 1);
windStrength = lerp(windStrength, targetWind, 0.002); // очень плавно


// Влияние на растения
const plantSwayAngle = windStrength * 15; // максимум 15 градусов наклона
const leafWave = windStrength * 20; // амплитуда листьев


Уровни ветра:
Тишина (wind = 0–0.1):
* Листья еле шевелятся
* Фрукты стоят
* Иногда один лист упадёт
Слабый бриз (wind = 0.1–0.3):
* Листья тихо шелестят (wave слева направо)
* Фрукты чуть качаются (amplitude: 5px)
* Ветки слегка наклонены
Средний ветер (wind = 0.3–0.6):
* Ветки заметно наклоняются
* Фрукты качаются значительнее (amplitude: 20px)
* Листья "улетают" один за одним (drift по ветру)
Сильный порыв (wind = 0.6–0.9):
* Ветки гнутся сильно
* Фрукты качаются (amplitude: 40px)
* Через 5–10 сек: один фрукт отрывается и улетает
Шторм (wind = 0.9–1.0):
* Редкий — 1 раз за блок, 30 сек
* Ветки гнутся максимально
* Несколько фруктов падают
* Потом тишина — контраст
Детальный сценарий блока (5 мин):
0:00–1:00: Тишина. Фрукты стоят. 1:00–1:30: Первый бриз появляется. Листья реагируют. 1:30–2:30: Ветер усиливается. Фрукты качаются. 2:30–3:00: Сильный порыв — ОДИН ФРУКТ отрывается!
* Фрукт: резкое ускорение → arc полёт → приземление с bounce
* Лежит на земле — качается и успокаивается 3:00–4:00: Ветер стихает. Постепенно тишина. 4:00–5:00: Полная тишина. Упавший фрукт лежит. Медленно.
Анимация ветра на листьях:
// Wave через крону дерева
const leafWave = (leafIndex, frame, windStrength) => {
  const waveOffset = leafIndex * 0.3; // задержка между листьями
  const baseWave = Math.sin((frame * 0.05 + waveOffset)) * windStrength * 20;

// При резком порыве: overshoot (перегиб) потом возврат


Музыка:
* Тишина: звук природы (птицы, тишина леса)
* Слабый ветер: мягкий вой через камышовую флейту
* Сильный ветер: нарастающий гул
* Момент падения фрукта: короткий акцент — потом тишина
* BPM 58 — синхронизировано с медленными порывами
