Duration: ~22 minutes | Language: Bahasa Indonesia (vocals) + English subtitles Age: 0–3 years | Format: Remotion Purpose: Entertainment + travel + geography — classic Indonesian train nursery rhyme

📋 REMOTION IMPLEMENTATION NOTES
Components needed
* TrainCharacter — engine with face, wheels spinning, steam from chimney
* TrainWagon — passenger wagon, windows with characters visible
* TrainWheel — rotating wheel animation synced to train speed
* SteamPuff — expanding fading circle from chimney/whistle
* SceneryScroll — parallax landscape scrolling (3 layers)
* StationBackground — platform, benches, flowers, signage
* MapDisplay — Java island map with route animation
* PassengerWindow — character visible in wagon window
* TunnelEffect — screen darkens with echo audio effect
Train wheel rotation
// Wheels rotate proportional to train speed
const trainSpeed = interpolate(frame, [0, 60], [0, maxSpeed], {
  extrapolateRight: 'clamp'
});
const wheelRotation = (frame * trainSpeed * 0.3) % 360;


Parallax scenery scroll
const layers = [
  { speed: 0.5, content: 'sky_clouds' },
  { speed: 1.0, content: 'mountains' },
  { speed: 2.0, content: 'trees_buildings' },
  { speed: 3.5, content: 'platform_ground' },
];
layers.map(layer => ({
  x: -(frame * layer.speed) % screenWidth,
}));


TUT whistle animation
const tutFrames = [120, 138, 156]; // three TUT moments
const activeTut = tutFrames.findIndex(f => frame >= f && frame < f + 12);
if (activeTut >= 0) {
  const tutProgress = (frame - tutFrames[activeTut]) / 12;
  // Steam puff expands
  const puffScale = interpolate(tutProgress, [0, 1], [0, 3]);
  const puffOpacity = interpolate(tutProgress, [0, 0.3, 1], [1, 0.8, 0]);
}

Audio design
* Song BPM = 105 → march tempo → 1 beat = 17 frames
* TUT WHISTLE: sharp, satisfying, 3 in sequence
* CHUGGA CHUGGA: rhythmic loop, tempo matches song BPM
* Tunnel echo: reverb effect on all sounds inside tunnel
* Night version: same music but softer, slightly slower
* Station ambiance: crowd murmur, announcements (soft)
* Suling (bamboo flute) for outro: distinctly Indonesian
