📋 REMOTION IMPLEMENTATION NOTES
Components needed
* ShapeDrawing — animated self-drawing shape using SVG stroke-dashoffset trick
* ShappyCharacter — main round character, idle/excited/dance states
* RealWorldScene — background scene with labeled objects
* ShapeCompare — side-by-side comparison with arrows
* GameScreen — 4-shape quiz layout with wrong/right states
* CelebrationOverlay — confetti + star particles
* ShapeParade — all shapes marching across screen
* SongScreen — lyrics + dancing shapes synced to music
Shape drawing animation (Remotion)
// Animate stroke-dashoffset to 0 over ~30 frames
const progress = interpolate(frame, [0, 30], [1, 0]);
const strokeDashoffset = perimeter * progress;


Timing system
30fps, 1920×1080. All animations via useCurrentFrame() + interpolate().
Audio layers
1. Background music (looped, ~20% volume)
2. Narrator voice (main)
3. SFX — drawing sounds, boings, sparkles, cheers
Color palette

* Rectangle: #FF8C00 (orange)

* Oval: #9B59B6 (purple)
