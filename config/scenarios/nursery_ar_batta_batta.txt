Water ripple effect
// Expanding rings on splash/quack
const ripples = splashEvents.map(event => {
  const age = frame - event.frame;
  const radius = age * 3;
  const opacity = interpolate(age, [0, 40], [0.8, 0], { extrapolateRight: 'clamp' });
  return { cx: event.x, cy: event.y, r: radius, opacity };
});

Fonts

Audio layers
1. Song track (Arabic vocals) — main
2. Background music — 20% volume
3. Sound effects — quack, splash, water, comedy sounds
4. Ambient pond sounds — birds, water — 10% volume
