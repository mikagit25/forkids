⚡ SHORTS-ВОРОНКА — План нарезки из готовых видео
Формат: вертикальный 9:16 (1080×1920) | Длина: 15–60 сек Цель: бесплатный трафик → подписки → просмотры длинных видео
________________


СТРАТЕГИЯ
Shorts для детского канала решают главную проблему нового канала — открытие (discovery). Алгоритм Shorts показывает контент людям которые НЕ подписаны. Родитель видит Short → нравится → открывает канал → включает длинное видео ребёнку.
Частота: 1-2 Shorts в день. Это легко — всё нарезается из готовых рендеров.
Правило детских Shorts: концепция понятна за 1 секунду, визуально чисто, мгновенный крючок.
________________


ТИП 1 — МОМЕНТЫ-КРЮЧКИ (из готовых сценариев)
Short 1.1 — HAP! (из Cicak)
* Кадр: ящерица замерла, комар приближается
* 3 сек напряжение → HAP! язык → поймал → victory dance
* Луп: 15 сек, зацикленный — комар появляется снова
* Текст-оверлей: "Wait for it... 🦎"
* Звук: тишина → HAP! → фанфара
Short 1.2 — DOR! (из Balonku)
* Зелёный шарик надувается... надувается... тишина 2 сек... DOR!
* Конфетти → новый шарик появляется → луп
* 20 сек цикл
* Это сатисфайнг-механика — люди смотрят 5+ лупов
Short 1.3 — Яйцо вылупляется (из Dajaja)
* Яйцо трясётся → трещина → ещё → ЦЫПЛЁНОК! PEEP!
* 5 яиц подряд — каждое своего цвета
* 45 сек — каждое вылупление = микро-награда
Short 1.4 — Поезд в туннель (из Naik Kereta)
* Поезд едет → туннель → ТЕМНОТА (глаза поезда светятся) → выезд в горный пейзаж
* WHOOSH-эффект — 20 сек луп
Short 1.5 — Радуга рисуется (из Pelangi)
* Гигантская кисть рисует радугу цвет за цветом
* Каждый цвет = звуковая нота вверх
* Полная радуга = аккорд + блеск — 25 сек
Short 1.6 — Пузыри лопаются (из сенсорных)
* Пузыри плывут вверх → лопаются по очереди → POP POP POP
* Переливающиеся цвета — 30 сек луп
Short 1.7 — Newton's cradle (из Pendulum)
* Металлические шары — клик-клак-клик-клак
* Гипнотический сатисфайнг — 30 сек идеальный луп
Short 1.8 — Маятниковая волна (из Pendulum loop 8)
* 9 разноцветных маятников — волновой узор
* Узор расходится → хаос → снова сходится — 60 сек
* Это вирусный физический контент — работает и на взрослых

ОФОРМЛЕНИЕ SHORTS
Обязательные элементы:
1. Первая секунда = крючок — движение или напряжение сразу
2. Текст-оверлей сверху — крупный, читается без звука (60% смотрят без звука)
3. Луп без шва — последний кадр = первый кадр
4. Конец = призыв: "Full video on our channel 👆" (последние 2 сек)
Технические параметры:
Разрешение: 1080×1920 (9:16)
FPS: 30
Длина: 15-60 сек (оптимум 20-35)
Битрейт: 10-12 Mbps


FFmpeg конвертация из горизонтального:
# Вариант 1: кроп центра
ffmpeg -i horizontal.mp4 -vf "crop=ih*9/16:ih,scale=1080:1920" short.mp4


# Вариант 2: блюр-подложка (видео целиком + размытые края)
ffmpeg -i horizontal.mp4 -vf "split[a][b];[a]scale=1080:1920,boxblur=20[bg];[b]scale=1080:-2[fg];[bg][fg]overlay=(W-w)/2:(H-h)/2" short.mp4


# Но ЛУЧШЕ: рендерить Shorts отдельно в Remotion с composition 1080×1920


Remotion отдельная композиция:
// В remotion.config — добавить вертикальную композицию
<Composition
  id="ShortHapMoment"
  component={HapMomentVertical}
  durationInFrames={450}  // 15 сек
  fps={30}
  width={1080}
  height={1920}
/>
// Перекомпоновать элементы под вертикаль — не кропать!
