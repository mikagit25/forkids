🍎 ТАНЦУЮЩИЕ ФРУКТЫ И ОВОЩИ — Двухэтапная серия
Формат: Remotion | Возраст: 0–3 года Этап 1: Без слов — только движение и музыка (визуальное знакомство) Этап 2: С текстом и озвучкой — глубокое изучение одного продукта
________________


ЛОГИКА ДВУХ ЭТАПОВ
Этап 1 — Знакомство через движение: Ребёнок просто видит продукт много раз в разных ситуациях. Мозг запоминает форму, цвет, характер движения. Никакого давления — нет вопросов, нет ответов. Как дети учатся в реальной жизни: сначала просто видят предмет.
Этап 2 — Изучение с названием: Ребёнок УЖЕ знает как выглядит яблоко — теперь учим слово. Тот же продукт, те же движения — но добавляем название + озвучку. Лёгкий диалог в конце — ребёнок "отвечает" на знакомый предмет.
________________

ЭТАП 1 — БЕЗ СЛОВ (серия A)
Правила Этапа 1:
* НОЛЬ слов — ни нарратора, ни текста
* Только музыка (инструментальная)
* Продукт танцует / движется / трансформируется
* Фон нейтральный: тёмный или белый
* Характер движения = характер продукта (яблоко — бодрое, банан — гибкий, лимон — острый)
________________


ВИДЕО A-1 — ЯБЛОКО ТАНЦУЕТ (30 мин, без слов)
Shape of movement: круглое = rolling + bouncing
Структура (6 блоков × 5 мин):
Блок 1 (0–5 мин): ПОЯВЛЕНИЕ И ЗНАКОМСТВО
* Тёмный фон
* Из центра медленно вырастает красное яблоко
* Поворачивается медленно на 360° — показывает все стороны
* Листик на хвостике шевелится от ветра
* Очень медленно — 5 сек на полный оборот
* BPM 75, тихая нежная музыка
Блок 2 (5–10 мин): ПРЫЖКИ
* Яблоко начинает прыгать — BOB
* Каждый прыжок: squash при приземлении, stretch при взлёте
* Ритм: 2 сек/прыжок
* Листик болтается при каждом прыжке
* Тень под яблоком появляется/исчезает
Блок 3 (10–15 мин): КАЧАНИЕ
* Яблоко качается влево-вправо (SWAY)
* При наклоне: листик тянется в сторону
* Медленно — 3 сек в каждую сторону
* Хвостик рисует маленькую дугу в воздухе
Блок 4 (15–20 мин): КАТИТСЯ
* Яблоко медленно катится слева направо
* При качении чуть сплющивается снизу
* Достигает края — мягко отскакивает — катится обратно
* Листик смотрит назад при движении вперёд
Блок 5 (20–25 мин): МНОГО ЯБЛОК
* Появляются 3 яблока — все прыгают с задержкой (волна)
* Красное, зелёное, жёлтое (разные сорта — одна форма!)
* Прыгают по очереди: красный, зелёный, жёлтый, красный...
Блок 6 (25–30 мин): ФИНАЛЬНЫЙ DRIFT
* Одно яблоко медленно дрейфует по экрану
* Очень медленно — Lissajous путь
* Листик указывает в сторону движения как стрелка
* BPM снижается до 60 — успокоение
Характер яблока:
Бодрое, круглое, упругое — движения всегда с bounce-эффектом. При каждом контакте с воображаемым полом — мягкий squash.
________________


ВИДЕО A-2 — БАНАН ТАНЦУЕТ (30 мин, без слов)
Shape of movement: изогнутое = swing + pendulum
Особенность банана:
Банан ИЗОГНУТ — это его главная визуальная черта. Все движения подчёркивают изогнутость.
Блок 1: ПОЯВЛЕНИЕ
* Банан растёт из пятна жёлтого цвета
* Медленно разгибается из прямой линии в форму банана
* Оборачивается к камере — показывает себя
Блок 2: МАЯТНИК
* Банан висит за кончик (как будто держится)
* Качается как маятник
* Форма подчёркивает дугу движения
Блок 3: CATERPILLAR MOVE
* Банан ползёт как гусеница (изгибается волной)
* Передний кончик опускается → задний подтягивается → передний снова
* Очень органичное движение для изогнутой формы
Блок 4: SPIN + ГИРО
* Банан вращается вокруг своего центра
* При вращении изогнутая форма создаёт красивый паттерн
Блок 5: ТРИ БАНАНА ВЕЕР
* 3 банана расставлены как веер
* Открываются и закрываются
* Меняют направление — то вверх, то вниз
Блок 6: DRIFT
* Банан плавно движется по экрану
* При изменении направления — мягко изгибается чуть сильнее
* Как рыба в воде
________________


ВИДЕО A-3 — АПЕЛЬСИН ТАНЦУЕТ (30 мин, без слов)
Shape of movement: тяжёлое, круглое = slow roll + wobble
Блок 1: Апельсин с текстурой кожуры медленно поворачивается Блок 2: Медленные прыжки — как тяжёлый мяч, низкие но грузные Блок 3: Wobble — покачивается как шар заполненный водой Блок 4: Катится медленнее яблока — более тяжёлый Блок 5: Апельсин разрезается пополам — видны дольки — половинки разъезжаются Блок 6: Снова собирается — и прыгает
________________


ВИДЕО A-4 — КЛУБНИКА ТАНЦУЕТ (30 мин, без слов)
Shape of movement: лёгкое, острое снизу = tip-tap + bounce
Особенность клубники: треугольная форма — острая снизу. Движения основаны на балансировании на острие.
Блок 1: Клубника балансирует на острой нижней точке — покачивается Блок 2: Прыгает и приземляется всегда на острие — точность Блок 3: Листики-корона колышутся — как пышная прическа Блок 4: Семечки на поверхности светятся по очереди — как огоньки Блок 5: 3 клубники пляшут рядом — разного размера Блок 6: Клубника "ныряет" в воображаемую воду — круги на воде
________________


ВИДЕО A-5 — МОРКОВЬ ТАНЦУЕТ (30 мин, без слов)
Shape of movement: длинное конусообразное = whip + wave
Особенность моркови: длинная — движения по всей длине.
Блок 1: Морковь стоит вертикально — ботва колышется как волосы Блок 2: Качается из стороны в сторону — как маятник-палочка Блок 3: Волна по длине моркови — как хвост рыбы Блок 4: Морковь вращается вокруг своей оси — как пропеллер Блок 5: 3 морковки в ряд — синхронная волна Блок 6: Морковь растёт из земли — медленно появляется снизу
________________

// Squash and stretch для фруктов
const applySquashStretch = (frame, config) => {
  const { bounceHeight, period, squashAmount, stretchAmount } = config;
  const bouncePhase = (frame % (period * 30)) / (period * 30);
  const y = Math.abs(Math.sin(bouncePhase * Math.PI)) * bounceHeight;
  const isLanding = y < 5 && bouncePhase > 0.5;
  return {
    scaleX: isLanding ? 1 + squashAmount : 1 - stretchAmount * (y / bounceHeight),
    scaleY: isLanding ? 1 - squashAmount * 0.7 : 1 + stretchAmount * (y / bounceHeight),
    y,
  };
};


// Для Этапа 2: LabelAppear — текст появляется под продуктом
const LabelAppear = ({ text, visible, frame, startFrame }) => {
  const opacity = interpolate(frame, [startFrame, startFrame + 20], [0, 1], {
    extrapolateRight: 'clamp'
  });
  return visible ? (
    <text
      style={{ fontFamily: 'Nunito, sans-serif', fontSize: 64, fontWeight: 800 }}
      fill="white"
      textAnchor="middle"
      opacity={opacity}
    >
      {text}
    </text>
  ) : null;
};
