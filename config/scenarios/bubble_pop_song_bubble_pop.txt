ФОРМУЛА BABY SHARK (разбор почему 16 млрд просмотров)
1. Мелодия из 4-5 нот — ребёнок 1 года может "спеть"
2. Структура повторения — та же строка × 3 + финальная: "Baby shark doo-doo-doo-doo-doo-doo ×3, Baby shark!"
3. Семейная прогрессия — baby → mommy → daddy → grandma → grandpa (ребёнок знает что дальше = предвкушение)
4. Жесты-движения — каждый персонаж = свой жест руками (акула = ладони-челюсти)
5. Ускорение в конце — финальный куплет быстрее = кульминация
6. Слоги-заполнители — "doo doo doo" поётся раньше слов
НАША ПЕСНЯ: "BUBBLE POP!"
Почему пузыри:
* Универсальны для всех культур (нет локализационных проблем)
* Уже есть ассеты из сенсорных видео
* POP! = встроенный момент-награда (как DOR и HAP)
* Жест простой: ладони соединены → раскрываются = POP

МЕЛОДИЯ
Нот всего 5: C-D-E-G-A (пентатоника — нет "неправильных" нот)
"Little bub-ble" = C-C-D-D
"pop-pop-pop-pop-pop-pop" = E-E-G-G-E-E
Финальный "POP!" = A (самая высокая — кульминация)
BPM: 100 (куплеты 1-5), 120 (финал)


Пентатоника важна: любые ноты ребёнок "подпоёт" — звучит гармонично.
________________


ВИЗУАЛ ОСНОВНОГО ВИДЕО (3 мин)
Персонажи-пузыри:
* Little bubble: маленький, голубой, большие глаза
* Mommy bubble: средний, розовый, бантик
* Daddy bubble: большой, синий, усы
* Grandma bubble: средний, фиолетовый, очки
* Grandpa bubble: большой, зелёный, борода из пены
Структура куплета (30 сек):
1. Персонаж надувается из трубочки (5 сек)
2. Поёт свою строку — подпрыгивает на каждый "pop" (15 сек)
3. Тишина 1 сек... POP! — лопается в конфетти (3 сек)
4. Конфетти складывается в следующего персонажа (5 сек)
Финал (30 сек):
* ВСЕ 5 пузырей на экране одновременно
* Поют вместе — темп растёт
* POP-POP-POP — все лопаются по очереди слева направо
* Экран в конфетти
* Из конфетти — снова маленький пузырь — подмигивает — луп!
Жест (показывается в углу всё видео):
Ладони вместе шариком → на "POP!" раскрываются + хлопок

СТРАТЕГИЯ ЗАПУСКА
1. Shorts первыми: 15-сек версия (один куплет + POP) за неделю до полного видео
2. Полное 3-мин видео — основной релиз
3. 20-мин расширенная — через неделю (для watch time)
4. 3 языковые версии — одновременно на всех рынках
5. Если песня цепляет → делать сиквелы: "Bubble Pop 2: Animals" (cat bubble, dog bubble...)
ПРОДАКШН
