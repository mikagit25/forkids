* StarCluster — stars in rainbow formation for outro
Rainbow drawing animation
// Draw each arc progressively
const RAINBOW_COLORS = [
  '#FF0000', // merah - red
  '#FF7F00', // oranye - orange
  '#FFFF00', // kuning - yellow
  '#00BB00', // hijau - green
  '#0000FF', // biru - blue
  '#4B0082', // nila - indigo
  '#8B00FF', // ungu - purple
];


// Each arc slightly smaller radius
RAINBOW_COLORS.map((color, i) => {
  const radius = 380 - i * 35;
  const startAngle = Math.PI; // left end
  const endAngle = 0; // right end


  // Draw progress
  const arcProgress = interpolate(
    frame,
    [i * 5, i * 5 + 30],
    [0, 1],
    { extrapolateRight: 'clamp', easing: Easing.out(Easing.cubic) }
  );


  return drawArc(cx, cy, radius, startAngle, endAngle * arcProgress, color, 30);
});

Audio design
* Song BPM = 85 — gentle, flowing, not bouncy
* Each rainbow color: unique soft musical note (C major scale)
* Rainbow appearance: harp glissando up
* Rainbow fade: harp glissando down
* Background: soft nature sounds throughout
* Night section: crickets + owl + soft wind
Emotional arc
Wonder (intro) → Learning colors (round 1) → Joy exploring (bridge 1) → Beauty appreciation (round 2) → Peace (calm) → Active participation (round 3) → Adventure (bridge 2) → Magic (night round) → Gratitude (finale) → Peaceful farewell (outro)
