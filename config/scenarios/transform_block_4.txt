ВИДЕО 4.1 — КАЛЕЙДОСКОП ИЗ ФРУКТОВ
Duration: 25 min | Loop: 4 min × 6 | BPM: 72
Концепция
Фрукты расставлены симметрично и вращаются. Паттерн как в настоящем калейдоскопе — 6-кратная симметрия. Бесконечно красиво и гипнотично.
Принцип симметрии:
* Экран делится на 6 секторов (60° каждый)
* В каждом секторе — один и тот же фрукт
* Все 6 копий вращаются вокруг центра
* Дополнительно: каждый фрукт вращается вокруг себя
// 6-кратная симметрия
const SECTORS = 6;
const BASE_ANGLE = (2 * Math.PI) / SECTORS;


const getKaleidoscopePositions = (baseX, baseY, frame) => {
  return Array.from({length: SECTORS}).map((_, i) => {
    const sectorAngle = i * BASE_ANGLE;
    const orbitAngle = (frame / (orbitPeriod * 30)) * 2 * Math.PI + sectorAngle;
    return {
      x: cx + Math.cos(orbitAngle) * orbitRadius,
      y: cy + Math.sin(orbitAngle) * orbitRadius,
      selfRotation: (frame / (selfRotPeriod * 30)) * 360,
    };
  });
};


Структура блоков:
Блок 1: ЯБЛОКИ × 6 (одинаковые)
* 6 красных яблок в симметрии
* Медленно вращаются по кругу (15 сек/оборот)
* Каждое яблоко также вращается вокруг себя
Блок 2: 6 РАЗНЫХ ФРУКТОВ
* 6 фруктов разного цвета и формы
* Та же орбита — но разнообразие форм создаёт богатый паттерн
Блок 3: ДВОЙНАЯ ОРБИТА
* 6 фруктов на внешней орбите (большой круг)
* 6 меньших фруктов на внутренней орбите
* Вращаются в противоположных направлениях
Блок 4: УСКОРЕНИЕ И ЗАМЕДЛЕНИЕ
* Вращение ускоряется: 15 сек → 8 сек → 4 сек/оборот
* Потом замедляется обратно
* Визуальный эффект "карусель разгоняется"
Блок 5: МОРФИНГ ОРБИТЫ
* Круговая орбита → эллиптическая → восьмёрка → снова круговая
* Все 6 фруктов следуют по изменяющейся траектории
Блок 6: ФИНАЛЬНЫЙ (НОЧНОЙ)
* Тёмный фон
* Фрукты оставляют светящиеся следы
* Следы образуют паттерн — след от калейдоскопа
* Очень красиво
Музыка:
* Циклическая, этно-мотивы
* Нота меняется с каждым оборотом (1/6 гаммы)
* Полный оборот = полная гамма

ВИДЕО 4.2 — ЗЕРКАЛО
Duration: 25 min | Loop: 4 min × 6 | BPM: 68
Концепция
Экран разделён вертикально пополам. Левая половина: оригинальные фрукты. Правая половина: их зеркальное отражение. Всё что делает оригинал — отражение повторяет зеркально.
Визуал:
// Зеркальное отражение
const mirrorX = (originalX) => screenW - originalX;


// Для каждого фрукта на левой стороне:
// оригинал в позиции (x, y)
// отражение в позиции (mirrorX(x), y)
// отражение: scaleX *= -1 (горизонтальный флип)


Структура блоков:
Блок 1: ОДНО ЯБЛОКО — ПРОСТОЕ ЗЕРКАЛО
* Яблоко слева делает BOB
* Отражение справа зеркально bobает
* Тонкая разделительная линия (1px, полупрозрачная)
Блок 2: ЯБЛОКО ТАНЦУЕТ — СЛОЖНЫЕ ДВИЖЕНИЯ
* Яблоко делает SWAY + BOB + небольшое вращение
* Отражение повторяет всё зеркально
* Красивый танец пары
Блок 3: ТРИ ФРУКТА СЛЕВА
* Три фрукта на левой половине со смещёнными фазами
* Шесть фруктов суммарно
* Богатый симметричный паттерн
Блок 4: ГОРИЗОНТАЛЬНОЕ ЗЕРКАЛО
* Теперь зеркало горизонтальное (верх-низ)
* Фрукты в верхней половине и их отражения снизу
Блок 5: ЧЕТЫРЕ ЗЕРКАЛА
* Два зеркала: вертикальное + горизонтальное
* 4 копии каждого фрукта (четыре квадранта)
* Очень богатый паттерн
Блок 6: ЗЕРКАЛО ДВИЖЕТСЯ
* Линия зеркала медленно перемещается слева направо
* Пропорция оригинал/отражение меняется
Музыка:
* Зеркальная: правый канал = левый канал
* При пересечении оси симметрии: нота отражается в другую октаву

ВИДЕО 4.3 — ХАОС → ПОРЯДОК
Duration: 25 min | Loop: 5 min × 5 | BPM: 75
Концепция
Фигуры/фрукты летают хаотично. Медленно — незаметно для ребёнка — выстраиваются в красивый паттерн. МОМЕНТ ОСОЗНАНИЯ: "Они сложились в цветок!" Потом снова хаос.
Детальный цикл (5 мин):
Фаза 1 — ХАОС (0–90 сек)
* 8 кругов летают по экрану
* Каждый по своей случайной траектории
* Не сталкиваются (мягкое отталкивание при сближении)
* Скорость: 2–4 px/сек
* Визуально: хаотичный полёт
Фаза 2 — СКРЫТЫЙ ПОРЯДОК НАЧИНАЕТСЯ (90–180 сек)
* Невидимо для ребёнка: начинают двигаться к целевым позициям
* Их случайное движение + drift к цели
* Всё ещё выглядит хаотично
* Целевые позиции: правильный восьмиугольник
Фаза 3 — ПАТТЕРН ПРОЯВЛЯЕТСЯ (180–240 сек)
* Уже видно что что-то складывается
* Фигуры почти на своих местах — но ещё двигаются
Фаза 4 — МОМЕНТ ПОРЯДКА (240–270 сек)
* Все фигуры встали на места — идеальный восьмиугольник
* Пауза 5 сек — тишина
* Все фигуры слегка светятся (пульсируют ±5%)
Фаза 5 — СНОВА ХАОС (270–300 сек)
* Фигуры резко "взрываются" в случайных направлениях
* Музыкальный акцент
* Снова хаотичное движение
Паттерны (каждый блок — другая форма):
Блок

	2
	Цветок (лепестки)
	3

// Каждая фигура: хаотическое движение + drift к цели
const chaosX = Math.sin(frame * chaos.freqX + chaos.phaseX) * chaos.ampX;
const chaosY = Math.cos(frame * chaos.freqY + chaos.phaseY) * chaos.ampY;


// Прогресс упорядочения 0→1 за 150 сек
const orderProgress = interpolate(frame, [chaosEnd, orderStart], [0, 1], {
  easing: Easing.inOut(Easing.cubic),
  extrapolateRight: 'clamp',
});


// Финальная позиция = смесь хаоса и порядка
const finalX = lerp(
  chaos.baseX + chaosX,  // хаотичная позиция
  target.x,              // целевая позиция
  orderProgress
);

ВИДЕО 4.4 — СОРТИРОВКА ПО РАЗМЕРУ
Duration: 25 min | Loop: 4 min × 6 | BPM: 72
Концепция
Фрукты перемешаны в случайном порядке. Медленно сортируются по размеру (маленький → большой). Потом снова перемешиваются. Satisfying цикл.
Детальный цикл (4 мин):
Фаза 1 — СЛУЧАЙНЫЙ ПОРЯДОК (0–30 сек)
* 5 яблок разного размера стоят в ряд
* Размеры: 60px, 100px, 80px, 140px, 40px (случайно)
* Все медленно BOB
Фаза 2 — НАЧАЛО СОРТИРОВКИ (30–180 сек)
Алгоритм пузырьковой сортировки — visually perfect!
Каждое сравнение:
* Два соседних яблока "смотрят" друг на друга (поворачиваются)
* 2 сек: сравниваем (оба чуть сжимаются — "думают")
* Если нужно менять местами: каждое поднимается вверх на 80px по дуге
* Меняются местами
* Опускаются обратно
* Следующая пара
Скорость: каждый обмен = 4 сек
Фаза 3 — ОТСОРТИРОВАНО (180–210 сек)
* Все в порядке: 40 → 60 → 80 → 100 → 140
* Пауза — все синхронно подпрыгивают (радость!)
* Маленькое конфетти (несколько кружков разлетаются)
Фаза 4 — ПЕРЕМЕШИВАНИЕ (210–240 сек)
* Яблоки разлетаются в случайных направлениях
* Приземляются в новом случайном порядке
* Снова не отсортированы

	Алгоритм сортировки (пузырьковый — визуально понятный):
// Состояние сортировки
type SortState = {
  array: Fruit[];
  step: 'compare' | 'swap' | 'move';
  i: number; // текущий индекс сравнения
  swapping: boolean;
  swapProgress: number; // 0-1
};


// Анимация обмена
const swap = (a: Fruit, b: Fruit, progress: number) => {
  // a летит по дуге вверх-вправо
  const aX = lerp(a.originalX, b.originalX, progress);
  const aY = a.y - Math.sin(progress * Math.PI) * 80; // дуга


  // b летит по дуге вниз-влево (под a)
  const bX = lerp(b.originalX, a.originalX, progress);
  const bY = b.y + Math.sin(progress * Math.PI) * 30; // нижняя дуга


  return { aX, aY, bX, bY };
};


Музыка:
* При каждом сравнении: две разные ноты (высокая и низкая)
* При обмене: мелодичный свист
* При завершении сортировки: маленькая fanfare (3 ноты вверх)
* При перемешивании: descending глиссандо

// Симметрия (для калейдоскопа)
const applyNSymmetry = (pos, center, n) => {
  return Array.from({length: n}).map((_, i) => {
    const angle = (i / n) * 2 * Math.PI;
    return rotateAround(pos, center, angle);
  });
};
