ВИДЕО 1.1 — ФРУКТ РАСТЁТ
Duration: 25 min | Loop: 5 min × 5 | BPM: 65
Концепция
Полный цикл жизни фрукта: семечко → росток → дерево → плод → падение → семечко. Самый медленный и успокаивающий сценарий блока. Каждый полный цикл = 2.5 минуты. Два цикла на блок.
Фрукты в ротации (каждый блок — другой фрукт):
* Блок 1: Яблоко
* Блок 2: Апельсин
* Блок 3: Клубника
* Блок 4: Банан
* Блок 5: Смешанный — все по очереди быстро
Детальный цикл (2.5 мин):
Фаза 1 — СЕМЕЧКО (0–20 сек)
* Тёмный фон (#0A1628)
* В центре: крошечная коричневая точка
* Медленно набухает: 5px → 15px за 10 сек
* Появляется маленький белый корешок вниз
* Корешок растёт: уходит вглубь экрана
Фаза 2 — РОСТОК (20–50 сек)
* Из семечка вверх медленно появляется зелёная линия-стебель
* 1px/сек скорость роста
* На конце: крошечный свёрнутый листик
* Листик медленно разворачивается
* Второй листик появляется чуть ниже
Фаза 3 — МОЛОДОЕ ДЕРЕВЦЕ (50 сек–1:20)
* Стебель утолщается: 2px → 8px за 30 сек
* Ветки расходятся в стороны
* Листья появляются на каждой ветке
* Дерево заполняет верхние 60% экрана
Фаза 4 — ПЛОД ПОЯВЛЯЕТСЯ (1:20–1:50)
* На одной из веток появляется крошечная точка нужного цвета
* Медленно растёт: точка → горошина → фрукт нормального размера
* Занимает 30 секунд
* Фрукт покачивается на ветке (BOB медленный)
* Цвет фрукта становится насыщеннее — зреет
Фаза 5 — ПАДЕНИЕ (1:50–2:10)
* Ветка слегка сгибается под весом
* Фрукт отрывается — медленно
* Падает с физикой: ускорение, squash при приземлении
* Лежит на земле внизу экрана — покачивается
Фаза 6 — СЕМЕЧКО ВЫХОДИТ (2:10–2:30)
* Фрукт медленно "тает" — уменьшается
* Из него появляется семечко
* Семечко падает в землю
* Земля закрывается над семечком
* Пауза 3 секунды — тишина
* Снова фаза 1

Музыка:
Одна нота каждые 4 секунды — как капли воды. BPM 65. Тишина — главный звук этого видео.

ВИДЕО 1.2 — ФИГУРА МЕНЯЕТ ФОРМУ (МОРФИНГ)
Duration: 25 min | Loop: 3 min × 8 | BPM: 70
Концепция
Одна фигура плавно перетекает из одной формы в другую. Без остановок — непрерывная трансформация. Это самое гипнотичное видео блока — форма всегда меняется, никогда не стоит.
Цикл форм (каждый проход 30 сек):
КРУГ → ОВАЛ → ПРЯМОУГОЛЬНИК → КВАДРАТ → ТРЕУГОЛЬНИК → ЗВЕЗДА → СЕРДЦЕ → КРУГ
Детальный цикл:
Круг → Овал (0–30 сек)
* Круг стоит 5 сек — покачивается (BOB)
* Начинает вытягиваться по горизонтали
* scaleX: 1.0 → 2.0 за 20 сек (easing inOut)
* scaleY: 1.0 → 0.6 за 20 сек
* Держится как овал 5 сек
Овал → Прямоугольник (30–60 сек)
* Скругления углов уменьшаются
* rx: большой → 0 за 20 сек
* Форма становится прямоугольной
Прямоугольник → Квадрат (60–90 сек)
* Ширина уменьшается до равной высоте
* Все 4 угла равны
Квадрат → Треугольник (90–120 сек)
* Нижние углы остаются на месте
* Верхние два угла медленно сходятся в одну точку
* Path morphing через SVG
Треугольник → Звезда (120–150 сек)
* Из углов треугольника вырастают дополнительные точки
* 3 точки → 5 точек звезды
Звезда → Сердце (150–180 сек)
* Самый эффектный переход
* Звезда "сминается" с двух сторон
* Верх раздваивается
* Появляется форма сердца
Сердце → Круг (180–210 сек)
* Сердце скругляется
* Ямка сверху заполняется
* Нижняя точка втягивается
* Идеальный круг
Цвет при трансформации:
Цвет медленно меняется вместе с формой:
* Круг: синий
* Овал: сине-зелёный
* Прямоугольник: зелёный
* Квадрат: жёлто-зелёный
* Треугольник: жёлтый
* Звезда: оранжевый
* Сердце: красный
* Круг: снова синий
Вариации по блокам:
* Блок 1–2: одна большая фигура в центре (200px)
* Блок 3–4: три фигуры разного размера — все морфируют синхронно
* Блок 5–6: три фигуры — каждая с задержкой 15 сек (волна морфинга)
* Блок 7–8: медленнее в 2 раза — для засыпания
// SVG path morphing (упрощённый)
// Для каждой формы: массив точек в полярных координатах
const SHAPES = {
  circle:    Array.from({length:64}).map((_,i) => ({ r: 100, a: i/64*2*Math.PI })),
  star:      Array.from({length:64}).map((_,i) => ({
    r: i%2===0 ? 100 : 45,
    a: i/64*2*Math.PI
  })),
  // ...etc
};


// Interpolate between shapes
const morphed = SHAPES[from].map((pt, i) => ({
  r: lerp(pt.r, SHAPES[to][i].r, progress),
  a: lerp(pt.a, SHAPES[to][i].a, progress),
}));


Музыка:
Плавный синтезатор-пад. Тональность меняется с каждой формой. Переход: глиссандо вверх при каждой смене формы.

ВИДЕО 1.3 — БОЛЬШОЙ И МАЛЕНЬКИЙ (пульсация размера)
Duration: 25 min | Loop: 4 min × 6 | BPM: 60
Концепция
Фрукты пульсируют между гигантским и крошечным размером. Разные скорости и фазы создают живой органичный паттерн. Самое медитативное видео блока — монотонное и предсказуемое.
Структура блоков:
Блок 1 (0–4 мин): ОДИНОКИЙ ПУЛЬС
* Одно большое яблоко в центре
* Медленно растёт: 150px → 250px за 4 сек
* Медленно уменьшается: 250px → 150px за 4 сек
* Цикл 8 сек — идеально синхронизирован с BPM 60
Блок 2 (4–8 мин): ДВА В ПРОТИВОФАЗЕ
* Яблоко (слева) растёт → клубника (справа) уменьшается
* Когда одно большое — другое маленькое
* Визуально: качели
Блок 3 (8–12 мин): ТРОЕ С ЗАДЕРЖКОЙ
* 3 фрукта — волна пульсации слева направо
* Задержка 1.5 сек
* Левый → средний → правый → левый
Блок 4 (12–16 мин): ХАОС → СИНХРОН
* 4 фрукта — каждый пульсирует в своём ритме (хаос)
* За 30 сек постепенно синхронизируются
* Все четыре ВМЕСТЕ: satisfying момент!
* 2 мин синхрона → снова расходятся в хаос
Блок 5 (16–20 мин): ОЧЕНЬ БОЛЬШОЙ → ОЧЕНЬ МАЛЕНЬКИЙ
* Один фрукт — экстремальный диапазон
* Маленький: 20px (едва виден)
* Большой: 400px (почти весь экран)
* Очень медленно: 8 сек на каждую крайность
Блок 6 (20–25 мин): ФИНАЛЬНЫЙ ПУЛЬС (замедление)
* Все фрукты вместе — синхронная пульсация
* Постепенно замедляется: 8 сек → 12 сек → 18 сек/цикл
* К концу почти стоят
// Пульсация с easing
const pulseScale = (frame, period, min, max, phase = 0) => {
  const t = (frame + phase) / (period * 30);
  const sin = Math.sin(t * 2 * Math.PI);
  const normalized = (sin + 1) / 2; // 0 to 1
  return min + (max - min) * Easing.inOut(Easing.sine)(normalized);
};


// Синхронизация хаоса
const syncProgress = interpolate(frame, [chaosEnd, syncStart], [0, 1]);
const effectivePhase = lerp(individualPhase, 0, syncProgress);


Музыка:
Одна нота пульсирует с каждым циклом размера. Громкость = размер фрукта. Большой → громче. Маленький → тише.

ВИДЕО 1.4 — ДЕЛЕНИЕ И СЛИЯНИЕ
Duration: 25 min | Loop: 5 min × 5 | BPM: 72
Концепция
Одна фигура делится на две, две на четыре, четыре на восемь. Потом все восемь снова сливаются в одну. Как клетки — биологически правдоподобно и завораживающе.
Детальный цикл (2.5 мин):
Фаза 1 — ОДИН (0–10 сек)
* Один большой синий круг в центре (200px)
* Медленно пульсирует (PULSE ±5%)
Фаза 2 — ДЕЛЕНИЕ 1→2 (10–30 сек)
* Круг медленно вытягивается по горизонтали (как клетка при делении)
* Образуется перетяжка в середине
* Перетяжка сужается: 200px → 2px
* РАЗРЫВ — два круга по 100px разлетаются влево и вправо
* Каждый пульсирует → растёт до 140px
Фаза 3 — ДЕЛЕНИЕ 2→4 (30–60 сек)
* Оба круга одновременно делятся — вверх и вниз
* Та же механика: вытягивание → перетяжка → разрыв
* Четыре круга по 80px расположены ромбом
Фаза 4 — ДЕЛЕНИЕ 4→8 (60–100 сек)
* Все четыре делятся одновременно
* Восемь кругов по 50px
* Расположены равномерно по экрану
* Все вместе пульсируют медленно
Пауза (100–120 сек)
* Восемь кругов просто плавают — очень медленный DRIFT
* Не уходят за край
* Музыка чуть тише
Фаза 5 — СЛИЯНИЕ 8→4 (120–150 сек)
* Восемь кругов начинают двигаться к четырём точкам
* Пары сближаются → касаются → СЛИВАЮТСЯ в один
* При слиянии: вспышка яркости на 0.5 сек
* Четыре круга по 80px
Фаза 6 — СЛИЯНИЕ 4→2 (150–165 сек)
* Четыре → два
Фаза 7 — СЛИЯНИЕ 2→1 (165–180 сек)
* Два → один
* Финальный круг: 200px, яркий
* Радостная пульсация 3 сек
* Цикл начинается снова
Цвет:
* 1 круг: насыщенный синий
* 2 круга: синий и сине-фиолетовый
* 4 круга: синий, зелёный, фиолетовый, голубой
* 8 кругов: радужный разброс цветов
При слиянии: цвета "смешиваются" — результирующий цвет = среднее.
Вариации по блокам:
* Блок 1: круги
* Блок 2: квадраты (деление выглядит иначе — как разрезание ножом)
* Блок 3: треугольники (деление пополам по высоте)
* Блок 4: смешанные фигуры
* Блок 5: фрукты! (яблоко делится — как будто режут)

* При делении: нота раздваивается (октава вниз)
* При слиянии: две ноты сходятся в унисон
* BPM 72 — средний темп, не быстрый

// Никогда резко — lerp через HSL
const lerpColorHSL = (from, to, t) => {
  const h = lerp(from.h, to.h, t);
  const s = lerp(from.s, to.s, t);
  const l = lerp(from.l, to.l, t);
  return `hsl(${h}, ${s}%, ${l}%)`;
};
