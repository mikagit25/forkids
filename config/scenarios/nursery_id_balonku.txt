🎈 Balonku Ada Lima — Full Video Script
Duration: ~22 minutes | Language: Bahasa Indonesia (vocals) + English subtitles Age: 0–3 years | Format: Remotion Purpose: Entertainment + learning colors — classic Indonesian balloon nursery rhyme
________________

🎬 INTRO — 0:00–1:00
Animation
* Blue sky background — sunny day — fluffy white clouds
* From bottom of screen — one by one — five balloons float up
* Each balloon has a smiling face:
   * 🟢 HIJAU (Green) — calm, cool expression
   * 🟡 KUNING (Yellow) — bright, sunny smile

Balloon float physics
// Natural floating — each balloon unique
const floatY = Math.sin(frame * 0.04 + balloonIndex) * 15;
const floatX = Math.cos(frame * 0.03 + balloonIndex * 0.7) * 8;
const rotation = Math.sin(frame * 0.05 + balloonIndex * 0.5) * 5;


Pop sequence
const popFrame = 245; // exact frame of DOR
const isPopped = frame > popFrame;
const popProgress = interpolate(frame, [popFrame, popFrame + 8], [0, 1], {
  extrapolateRight: 'clamp'
});
// Expand circle rapidly then disappear
const popScale = interpolate(popProgress, [0, 0.3, 1], [1, 2.5, 0]);
const popOpacity = interpolate(popProgress, [0, 0.3, 1], [1, 0.8, 0]);
