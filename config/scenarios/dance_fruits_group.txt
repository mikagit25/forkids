ГЛАВНЫЕ ПРАВИЛА СЕРИИ
1. Минимум 4–5 минут на один двигательный блок — без смены паттерна
2. Фрукты не исчезают между блоками — только меняют движение
3. Смена движения плавная — 30-секундный переход без резких изменений
4. Не более 6 фруктов одновременно на экране — не перегружать
5. Фон тёмный (#0A1628) — фрукты яркие на тёмном = максимальный контраст
6. BPM 70–85 — никогда не быстрее, это не дискотека
7. easing: inOut(sine) для ВСЕХ движений — никогда linear

ВИДЕО 1 — КРАСНАЯ СЕМЬЯ (яблоко, клубника, помидор)
Duration: 25 min | BPM: 75 | Mood: тёплое, радостное
Блок 1 (0:00–5:00): ПОЯВЛЕНИЕ ПО ОДНОМУ
0:00–1:20 — Яблоко появляется в центре-левой части
* Медленно вырастает из точки до полного размера (easing out elastic)
* Делает один медленный поворот 360° — показывает себя
* Начинает тихий BOB на месте
1:20–2:40 — Клубника появляется справа
* Та же механика: вырастает → показывает → начинает BOB
* Чуть другая фаза BOB — не синхронно с яблоком
2:40–4:00 — Помидор появляется в центре между ними
* Вырастает → поворот → BOB
* Теперь все трое в воздухе — каждый чуть на своей высоте
4:00–5:00 — Синхронизация
* Постепенно (за 30 сек) их BOB синхронизируется
* К концу блока все три прыгают ВМЕСТЕ — satisfying момент
Блок 2 (5:00–10:00): ВОЛНА
Три фрукта в линии начинают волновое движение. Левый (яблоко) → центр (помидор) → правый (клубника) → обратно. Задержка 0.35 сек между соседями.
Вариация 5:00–7:00: Волна слева направо Вариация 7:00–9:00: Волна справа налево Вариация 9:00–10:00: Волна от центра — помидор прыгает → яблоко и клубника одновременно
Визуально:
яблоко     помидор    клубника
  ↑            .            .    ← t=0
  .            ↑            .    ← t+0.35
  .            .            ↑    ← t+0.70
  .            ↑            .    ← t+1.05
  ↑            .            .    ← t+1.40  (цикл)


Блок 3 (10:00–15:00): ЗЕРКАЛО-РАЗДВИЖЕНИЕ
10:00–12:30: Яблоко и клубника медленно расходятся к краям. Помидор остаётся в центре. Потом сближаются обратно. Цикл 8 сек. Движение: horizontal lerp с easing inOut.
12:30–15:00: Все трое качаются (SWAY) влево-вправо синхронно. 3 сек в левую, 3 сек в правую — как один организм. В момент максимального отклонения — каждый немного поворачивается в сторону движения.
Блок 4 (15:00–20:00): МАЯТНИК-СИНХРОН
Все три качаются как маятники. Амплитуда: 60px в каждую сторону. Период: 4 сек туда, 4 сек обратно. ВСЕ ВМЕСТЕ — это самое гипнотичное движение всего видео.
Вариация 17:30–20:00: Маятник постепенно замедляется. Амплитуда уменьшается: 60px → 40px → 20px Скорость уменьшается: 4 сек → 5 сек → 6 сек К концу блока почти стоят — еле заметно качаются.
Блок 5 (20:00–23:00): ХОРОВОД-ТРЕУГОЛЬНИК
Три фрукта образуют треугольник и медленно вращаются по кругу. Центр вращения — центр экрана. Скорость: один полный оборот за 12 секунд. Каждый фрукт дополнительно делает BOB (мини).
Это создаёт красивый паттерн: фрукты вращаются по орбите + покачиваются.
Блок 6 (23:00–25:00): ФИНАЛЬНЫЙ DRIFT
Три фрукта начинают медленно дрейфовать по экрану — каждый по своему пути. Lissajous кривые — разные частоты → никогда не повторяются. Музыка тихает. Фрукты просто плавают.

ВИДЕО 2 — КРАСНАЯ СЕМЬЯ + ВИШНЯ (4 фрукта)
Duration: 25 min | BPM: 78 | Mood: весёлое
Те же яблоко, клубника, помидор + вишня (пара на ножке).
Блок 1 (0:00–5:00): ВИШНЯ ПРИХОДИТ В ГОСТИ
Первые 3 фрукта уже на экране из видео 1 (или появляются за 2 мин). Вишня появляется сверху — спускается медленно. Покачивается на ножке как качели. Три остальных СМОТРЯТ на неё (поворачиваются).
Блок 2 (5:00–10:00): ЧЕТЫРЕ В ЛИНИЮ — ВОЛНА
Четыре фрукта в линию. Волна. С 4 фруктами волна длиннее и красивее. Задержка 0.3 сек между соседями.
Особенность вишни: она — ПАРА двух ягод. При подъёме вишня качает обеими ягодками.
Блок 3 (10:00–15:00): КВАДРАТ-ХОРОВОД
Четыре фрукта встают в квадрат (углы). Медленно вращаются по часовой стрелке. Один оборот = 15 сек. Потом меняют направление — против часовой.
Блок 4 (15:00–20:00): ПАРНЫЙ ТАНЕЦ
Пара 1: Яблоко и клубника качаются навстречу друг другу (SWAY синхронный но противофазный — когда один вправо другой влево — они "общаются") Пара 2: Помидор и вишня делают то же самое
Потом пары меняются: яблоко+вишня, клубника+помидор
Блок 5 (20:00–23:00): БОЛЬШОЙ МАЯТНИК
Все четыре — единый маятник. Вся группа качается влево-вправо как один. Самый монотонный и успокаивающий блок.
Блок 6 (23:00–25:00): DRIFT
________________


ВИДЕО 3 — ЖЁЛТАЯ СЕМЬЯ (банан, лимон, кукуруза)
Duration: 25 min | BPM: 80 | Mood: солнечное, бодрое
Особенность этой группы:
Все три — вытянутые формы (не круглые). Это делает их движения интереснее. Банан изогнутый, лимон овальный, кукуруза длинная и прямая.
Блок 1 (0:00–5:00): ПОЯВЛЕНИЕ + ФОРМ-ШОУ
Каждый появляется и делает полный поворот 360°. После поворота — ребёнок видит: все жёлтые, все вытянутые, но разные.
Блок 2 (5:00–10:00): МАЯТНИКИ-РАЗНЫЕ
Банан качается как широкий маятник (большая амплитуда). Лимон качается быстрее (меньше — легче). Кукуруза почти не качается (длинная = инертная).
Все три качаются по-разному → красивый паттерн несинхронных маятников. Иногда случайно совпадают фазы — все три одновременно слева → satisfying!
Блок 3 (10:00–15:00): ВОЛНА + ВРАЩЕНИЕ
Волна слева направо — как обычно. НО: банан при подъёме ещё немного поворачивается (изогнутая форма меняется видом). Кукуруза при подъёме вытягивается вертикально — очень выразительно.
Блок 4 (15:00–20:00): ТРЕУГОЛЬНИК-ХОРОВОД
Хоровод треугольником. Один оборот 14 сек. Особенность: у кукурузы вертикальное положение — при вращении по кругу это очень заметно.
Блок 5 (20:00–23:00): СИНХРОННЫЙ SWAY
Все трое качаются влево-вправо ВМЕСТЕ. Медленно — 5 сек в каждую сторону. Абсолютно синхронно — как одна сущность.
Блок 6 (23:00–25:00): DRIFT
________________


ВИДЕО 4 — ЖЁЛТЫЕ + ОРАНЖЕВЫЕ (банан, ананас, апельсин, морковь)
Duration: 28 min | BPM: 82 | Mood: тропическое, яркое
Четыре продукта: два жёлтых (банан, ананас) + два оранжевых (апельсин, морковь).
Блок 2 (5:00–10:00): ДВА ХОРОВОДА
Левый хоровод: Банан + морковь вращаются по маленькому кругу (r=80px) Правый хоровод: Ананас + апельсин вращаются по своему кругу (r=80px) Оба вращаются в одну сторону — по часовой.
Потом оба меняют направление — против часовой.
Блок 3 (10:00–15:00): ВЗАИМНЫЙ ОБМЕН
Левый хоровод и правый хоровод медленно (за 20 сек) меняются местами. Продукты движутся по большой дуге через центр экрана. Это медленно и красиво.
Блок 4 (15:00–21:00): ЧЕТЫРЕ В КВАДРАТЕ
Четыре продукта в углах экрана (просторно). Все четыре маятник-синхрон — левые → правые → левые.
________________


ВИДЕО 5 — ЗЕЛЁНАЯ СЕМЬЯ (арбуз, брокколи, горох, огурец)
Duration: 28 min | BPM: 72 | Mood: спокойное, природное
Особенность:
Зелёные продукты самые "тяжёлые" визуально. Арбуз огромный, брокколи пышный, горох маленький, огурец длинный. Контраст размеров создаёт интересную динамику.
Блок 1 (0:00–5:00): КОНТРАСТ РАЗМЕРОВ
Арбуз — самый большой — появляется в центре. Горох — самый маленький — появляется рядом с ним. Брокколи и огурец добавляются.
Все прыгают вместе — но горох прыгает НАМНОГО ВЫШЕ (маленький, лёгкий). Арбуз почти не прыгает (тяжёлый) — только чуть покачивается.
Блок 2 (5:00–10:00): МАЯТНИК — БОЛЬШОЙ И МАЛЕНЬКИЙ
Арбуз: очень медленный маятник (6 сек в сторону) Огурец: средний (4 сек) Брокколи: средний (4 сек, но другая фаза) Горох: быстрый (2 сек) — маленький и лёгкий!
Они не синхронны → живой органичный паттерн.
Блок 3 (10:00–15:00): ГОРОХ ПРЫГАЕТ ВОКРУГ АРБУЗА
Арбуз стоит в центре — медленно покачивается. Горох прыгает вокруг него по маленькому кругу. Как планета вокруг звезды. Брокколи и огурец — в стороны, свои движения.
Блок 4 (15:00–21:00): ВМЕСТЕ — СИНХРОН
Все четыре синхронный SWAY. Медленно — BPM 72. Брокколи при наклоне пышная крона колышется дополнительно.

ВИДЕО 6 — ОРАНЖЕВАЯ СЕМЬЯ (апельсин, морковь, тыква)
Duration: 25 min | BPM: 76 | Mood: тёплое, уютное
Особенность:
Три круглых-почти-круглых продукта. Апельсин = идеальный шар. Тыква = большой, ребристый, медленный. Морковь = единственная вытянутая — контраст!
Блок 2 (5:00–10:00): ТЫКВА ДИРИЖИРУЕТ
Тыква в центре — покачивается медленно. Апельсин и морковь танцуют вокруг неё. Апельсин катится по дуге вокруг тыквы. Морковь качается рядом — длинная форма красиво движется.
Блок 3 (10:00–15:00): ТРИ МАЯТНИКА
Морковь: большая амплитуда наклона (вытянутая — видно наклон) Апельсин: средний маятник Тыква: еле-еле — самый тяжёлый
Все три в ОДНУ сторону = satisfying синхрон.
Блок 4 (15:00–20:00): ВОЛНА
Слева морковь — центр апельсин — справа тыква. Волна идёт → тыква реагирует медленнее (тяжёлая). Это физически правдоподобно и интересно!

ВИДЕО 7 — БОЛЬШОЙ ПАРАД (5 фруктов)
Duration: 28 min | BPM: 80 | Mood: праздничное, театральное
Яблоко, банан, морковь, арбуз, ананас. Самый разнообразный набор форм: круглые, вытянутые, большие, маленькие.
Блок 1 (0:00–5:00): ПАРАД-ВХОД
Продукты выходят на экран по одному с ЛЕВОЙ стороны. Идут как в параде — слева направо. Каждый со своим характером:
* Яблоко: бодрый прыжок при движении (bounce-walk)
* Банан: покачивается влево-вправо (swagger)
* Морковь: вертикальная — идёт прямо (гордо)
* Арбуз: медленно катится, немного вперевалочку
* Ананас: марширует — корона держится гордо
Блок 2 (5:00–10:00): ПАРАД ОСТАНОВИЛСЯ — ЛИНИЯ
Все пятеро в ряд. Встали. Смотрят. Начинается волна — яблоко первое. С 5 продуктами волна красивее всего в серии.
Блок 3 (10:00–16:00): БОЛЬШОЙ ХОРОВОД
Все пятеро встают в круг и вращаются. Один оборот = 15 сек. Арбуз самый большой — его видно всегда. Ананас — корона мелькает при вращении.
Блок 4 (16:00–22:00): МАЯТНИК СТЕНЫ
Все пятеро — единый маятник. Вся стена из 5 продуктов качается влево-вправо. 5 сек в левую, 5 сек в правую. Арбуз в центре — якорь. Остальные вокруг него.
Блок 5 (22:00–26:00): РАЗЛЁТ И СБОРКА
22:00–23:30: Все медленно расплываются в разные углы экрана. 23:30–25:00: Пауза — каждый на своём месте — медленный BOB. 25:00–26:00: Все плавно возвращаются в линию. Цикл повторяется.
Финал (26:00–28:00): DRIFT
________________


ВИДЕО 8 — ВСЕ ВМЕСТЕ (6 продуктов, главное видео серии)
Duration: 30 min | BPM: 78 | Mood: торжественное, полное
Яблоко, банан, апельсин, клубника, морковь, ананас. Шесть разных продуктов — три цветовые семьи — максимальное разнообразие.
Блок 1 (0:00–6:00): БОЛЬШОЙ ВХОД
Продукты появляются попарно:
* Первые: яблоко (слева) + банан (справа) — 0:00–1:00
* Вторые: апельсин (снизу-лево) + клубника (снизу-право) — 1:30–2:30
* Третьи: морковь (сверху-лево) + ананас (сверху-право) — 3:00–4:00
* К 4:30: все шестеро на экране
* 4:30–6:00: синхронизация BOB — постепенно все начинают прыгать вместе
Блок 2 (6:00–12:00): СЕТКА-ПУЛЬС
Шесть продуктов расставляются в сетку 3×2.
яблоко    апельсин    ананас
банан    клубника   морковь


Каждый PULSE с разной фазой — живой дышащий ковёр. Фазы: позиция[i][j] delay = (i * 3 + j) * 0.4 сек
Потом синхронно все вместе пульсируют — один ритм.
Блок 3 (12:00–18:00): ДВА ХОРОВОДА + ОБМЕН
12:00–14:00: Верхняя тройка (яблоко, апельсин, ананас) вращается по часовой. Нижняя тройка (банан, клубника, морковь) вращается против часовой.
14:00–16:00: Два хоровода меняются направлениями.
16:00–18:00: ОБМЕН — верхняя тройка и нижняя тройка меняются местами. Движение по дуге через центр. Медленно — 30 сек на полный обмен.
Блок 4 (18:00–24:00): БОЛЬШОЙ МАЯТНИК-СТЕНА
Шесть продуктов — в два ряда — качаются как одна стена. 6 сек влево, 6 сек вправо. Самый монотонный и гипнотичный блок всего видео.
Вариация 21:00: Верхний ряд и нижний ряд в противофазе. Верхние влево — нижние вправо. И обратно. Это создаёт волновое движение сверху вниз.
Блок 5 (24:00–28:00): ШЕСТИУГОЛЬНИК-ХОРОВОД
Все шесть встают в правильный шестиугольник (гексагон). Медленно вращаются все вместе — один оборот 18 сек. Это самое красивое движение всей серии.
Финал (28:00–30:00): ФИНАЛЬНЫЙ DRIFT
Все шесть расплываются по экрану. Каждый по своему пути. Никуда не спешат. Музыка тихает. Просто плавают. Последние 30 сек — почти тишина.

Структура компонентов
// GroupDance — главный оркестратор
interface GroupDanceProps {
  fruits: FruitConfig[];    // массив продуктов с позициями и размерами
  motionType: GroupMotion;  // текущий тип группового движения
  transitionProgress: number; // 0-1 переход между блоками
}


type GroupMotion =
  | { type: 'WAVE'; direction: 'ltr' | 'rtl' | 'center'; delay: number }
  | { type: 'ORBIT'; centerX: number; centerY: number; radius: number; period: number; clockwise: boolean }
  | { type: 'SYNC_SWAY'; amplitude: number; period: number; phase: 'same' | 'opposite' }
  | { type: 'PENDULUM_WALL'; amplitude: number; period: number }
  | { type: 'GRID_PULSE'; rows: number; cols: number; phaseDelta: number }
  | { type: 'DRIFT'; paths: LissajousConfig[] }
  | { type: 'PARADE'; speed: number; direction: 'ltr' | 'rtl' }
  | { type: 'EXPLODE_GATHER'; targetPositions: Position[]; phase: 'explode' | 'hold' | 'gather' };


// Переход между блоками — 30 сек

const WAVE_DELAY_SEC = 0.35; // задержка между соседями
const WAVE_PERIOD_SEC = 2.5; // период одного подъёма-опускания


const getWaveY = (fruitIndex: number, frame: number) => {
  const delayFrames = fruitIndex * WAVE_DELAY_SEC * 30;
  const adjustedFrame = frame - delayFrames;
  if (adjustedFrame < 0) return 0;
  return Math.sin(adjustedFrame * (2 * Math.PI) / (WAVE_PERIOD_SEC * 30)) * WAVE_AMPLITUDE;
};


Хоровод (ORBIT) — позиции на окружности
const getOrbitPosition = (index: number, total: number, frame: number, config: OrbitConfig) => {
  const baseAngle = (index / total) * 2 * Math.PI;
  const rotationAngle = (frame / (config.period * 30)) * 2 * Math.PI * (config.clockwise ? 1 : -1);
  const angle = baseAngle + rotationAngle;
  return {
    x: config.centerX + Math.cos(angle) * config.radius,
    y: config.centerY + Math.sin(angle) * config.radius * 0.6, // немного сплющиваем по вертикали
  };
};

const getGridPulse = (row: number, col: number, frame: number) => {
  const phaseDelay = (row * cols + col) * PHASE_DELTA * 30;
  const adjustedFrame = frame - phaseDelay;
  return 1 + Math.sin(adjustedFrame * (2 * Math.PI) / (PULSE_PERIOD * 30)) * PULSE_AMOUNT;
};
