# Happy Bear Kids — Roadmap

**Принцип:** Сделать один раз → запускать автоматически.  
Каждая фаза — рабочий продукт, не прототип.

---

## Фаза 1 — Качественные Assets (1-2 сессии)
> Фундамент. Плохой спрайт = плохое видео навсегда.

### 1.1 Спрайты персонажей
- [ ] Скачать 30-50 PNG с прозрачным фоном: Flaticon / FreeSVG / OpenClipart
- [ ] Категории: фрукты, животные, буквы A-Z, цифры 0-9, цвета
- [ ] Стандарт: 512×512 RGBA, единый стиль (мультяшный, яркий)
- [ ] Скрипт `scripts/download_sprites.py` — качает и раскладывает по папкам

### 1.2 Музыка
- [ ] Kevin MacLeod уже есть (20 треков) — **достаточно для старта**
- [ ] Разметить треки по темпу (BPM) в `config/music_index.yaml` — для подбора к теме

**Результат:** `assets/sprites/{fruits,animals,letters,numbers}/` — готовые спрайты

## Фаза 2 — Видео-движок (2-3 сессии)
> Починить то что есть, сделать надёжным.

### 2.1 Починить хореографию
- [ ] Убрать доминирование `grid_bounce` в шаблоне — сцены должны реально меняться
- [ ] Проверить `carousel`, `parade`, `zigzag` — что они видны в итоговом видео
- [ ] Добавить плавные переходы между сценами (fade 0.5s)

### 2.2 Text overlay для образовательного контента
- [ ] `TextOverlay` класс — показывает букву/слово/цифру поверх видео
- [ ] Шрифты детские (уже скачать в `assets/fonts/`)
- [ ] Анимация: появление + исчезновение текста в такт сцене

### 2.3 Озвучка (gTTS)
- [ ] `scripts/generate_voiceover.py` — генерирует MP3 из текста
- [ ] Поддержка EN + RU
- [ ] Кэширование: один раз сгенерировал слово "Apple" → переиспользуешь

**Результат:** `kids video --type dance --theme animals` и `kids video --type abc` работают

## Фаза 3 — Контент-конвейер (2-3 сессии)
> Один запуск → несколько готовых видео в очереди.

### 3.1 Типы видео
| Тип | Описание | Длина |
|-----|----------|-------|
| `dance` | Танцевальное, разные хореографии | 10-30 мин |
| `abc_song` | Буква → слово → картинка → звук | 5-8 мин |
| `counting` | Цифры 1-10/1-20 с объектами | 5-8 мин |
| `colors` | Цвета + предметы + слова | 5-8 мин |

### 3.2 Пакетная генерация
- [ ] `scripts/batch_generate.py --plan config/weekly_plan.yaml`
- [ ] `weekly_plan.yaml` описывает 7 видео на неделю (тип, тема, длина)
- [ ] Генерирует все видео в `output/queue/`

### 3.3 Умная публикация
- [ ] `scripts/publish_queue.py` — берёт из `output/queue/`, загружает по расписанию
- [ ] Авто-thumbnail из первого кадра видео
- [ ] Мета-данные из `config/playlists.yaml`

**Результат:** В воскресенье запустил → 7 видео запланированы на всю неделю

## Фаза 4 — Образовательный контент (3-4 сессии)
> Ценность для родителей → просмотры и подписки.

### 4.1 ABC Song
- [ ] 26 сцен (по букве), каждая 10-15 сек
- [ ] Буква большая в центре, рядом картинка (Apple для A)
- [ ] gTTS произносит: "A — Apple", потом звук буквы "Aaa"
- [ ] Фоновая музыка тихо под озвучкой

### 4.2 Counting 1-10
- [ ] Цифра + N объектов (3 яблока для числа 3)
- [ ] Озвучка "One, Two, Three..."
- [ ] Анимация: объекты появляются по одному

### 4.3 Colors
- [ ] Фон меняет цвет, персонаж того же цвета
- [ ] "Red! The apple is red!"

**Результат:** 3 полноценных образовательных формата с шаблонами
