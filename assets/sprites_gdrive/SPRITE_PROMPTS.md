# Sprite Generation Prompts
# Промты для генерации спрайтов-персонажей фруктов и овощей
# Стиль должен совпадать с уже готовыми спрайтами в fruits_cartoon/

---

## ГРУППА 1 — Фрукты (красный/желтый)
Сохранить как: watermelon, cherry, lemon, mango

```
Create a set of 4 cute kawaii cartoon fruit characters on a white background:
1. Watermelon slice - round, smiling, red inside with black seeds
2. Cherry - bright red, pair on a stem, cute face
3. Lemon - yellow, oval shape, happy face, tiny arms and legs
4. Mango - orange-yellow, chubby body, big eyes with rosy cheeks

Style: kawaii chibi cartoon, each character has: big round body, large expressive
eyes, small stubby arms and legs, happy smiling mouth, rosy cheek blushes,
simple flat cartoon shading. White background. Children's educational illustration.
Characters should match this reference style: round body, face with big eyes,
little arms and legs sticking out.
```

Children's educational illustration. Same consistent style for all 4 characters.

## ГРУППА 4 — Овощи (часть 1)
Сохранить как: carrot, broccoli, corn, tomato, pumpkin

```
Create a set of 5 cute kawaii cartoon vegetable characters on a white background:
1. Carrot - orange, long triangle body, green leafy top, happy smiling face
2. Broccoli - dark green, tree-like shape, round bumpy top, cute face
3. Corn - yellow, tall body, green husk leaves, big happy smile
4. Tomato - round red, small green leaves on top, chubby body, winking eye
5. Pumpkin - round orange, cute happy face, green stem on top

Style: kawaii chibi cartoon style, each character has a unique body shape matching
the vegetable, with added cute face (big eyes, smile, rosy cheeks), small arms
and legs. White background. Consistent children's educational illustration style.
All characters should look friendly and cheerful.
```

## ГРУППА 5 — Овощи (часть 2)
Сохранить как: cucumber, eggplant, pepper, potato, onion

```
Create a set of 5 cute kawaii cartoon vegetable characters on a white background:
1. Cucumber - long green oval body, lighter stripes, happy face
2. Eggplant - dark purple teardrop shape, green top, round eyes, big smile
3. Bell pepper (red) - round red with bumps, green stem, cheerful expression

```

## После генерации:
1. Загрузи изображения в Google Drive папку
2. Скажи Клоду — он скачает и обработает автоматически через extract_fruit_sprites.py
3. Скрипт вырежет каждого персонажа и уберёт фон
